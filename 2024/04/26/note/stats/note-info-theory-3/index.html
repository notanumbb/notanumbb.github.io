假设检验直观来说就是找到一个 n 维 (x1,...,xn)(x_1, ..., x_n)(x1​,...,xn​) 的集合 AnA_nAn​，记作接受域，表示如果我们抽样真实得到的样本 (x1,...,xn)(x_1, ..., x_n)(x1​,...,xn​) 在接受域中则接受 H0H_0H0​ ，反之拒绝 H0H_0H0​；类似地我们也可以定义拒绝域 AncA^c_nAnc​

接受域的选择会影响这个检验的错误率，我们主要关注：

  • 一类错误：如果 XXX​ 真的来自于 ppp，但是我们拒绝了 ppp 的概率，也就是 P(Anc)=1−P(An)P(A^c_n) = 1- P(A_n)P(Anc​)=1−P(An​)
  • 二类错误：如果 XXX 其实来自于 qqq，但是我们接受了 ppp 的概率，也就是 Q(An)Q(A_n)Q(An​)

当然接受域越大此时 P(An)P(A_n)P(An​) 越大，对应一类错误自然会越小；但是反之的 Q(An)Q(A_n)Q(An​) 也就是二类错误会越大；反之接受域越小，对应的一类错误会上升，而二类错误会下降；二者是 trade-off 的关系；

通常进行假设检验时，我们会倾向于优先控制一类错误不超过阈值 ε\varepsilonε，而 Chernoff-stein lemma 给出了所有满足一类错误不超过阈值的接受域中，能够使得二类错误最优的检验方法，和对应方法能够达到的二类错误下界

Chernoff-stein lemma：考虑存在两个可能的分布 p,qp, qp,q，通过 n 个 iidiidiid 样本 XiX_iXi​ 进行假设检验时，对于任意给定的 ε\varepsilonε，考虑所有满足一类错误不超过 ε\varepsilonε (即 P(Bn)>1−εP(B_n)>1-\varepsilonP(Bn​)>1−ε ) 的接受域 BnB_nBn​ ，其中可以得到的最低的二类错误概率为 βε\beta^\varepsilonβε, 则有：

lim⁡n→∞βε→2−nD(p∣∣q)\lim _{n\to \infty} \beta^\varepsilon \to 2^{-nD(p||q)} n→∞lim​βε→2−nD(p∣∣q)

且对数似然比就是一个能够趋向最优下界的判别方法 (asymptotically optimal)，对应的接受域可以表示为：

Bn={xn:1nlog⁡P(X)Q(X)≥D(p∣∣q)−δε}B_n = \left\{x_n: \frac{1}{n}\log \frac{P(X)}{Q(X)} \geq D(p||q) - \delta_\varepsilon\right\} Bn​={xn​:n1​logQ(X)P(X)​≥D(p∣∣q)−δε​}


证明

这里的证明分两步：

  • 先证明对数似然比的做法的二类错误下界满足上面的式子；
  • 然后证明所有控制一类错误率为 ε\varepsilonε 的接受域中，上面的对数似然比的做法是二类错误最优的；

我们先考虑对数似然比对应的接受域；

1nlog⁡p(X)q(X)=1nlog⁡∏p(xi)∏q(xi)=1n∑i=1nlog⁡p(xi)q(xi)⟶PEXlog⁡p(x)q(x)\begin{aligned} \frac{1}{n} \log \frac{p(X)}{q(X)} &= \frac{1}{n}\log \frac{\prod p(x_i)}{\prod q(x_i)} \\ &= \frac{1}{n} \sum _{i=1}^n \log \frac{p(x_i)}{q(x_i)} \\ &\stackrel{P}{\longrightarrow} E_X \log \frac{p(x)}{q(x)} \end{aligned} n1​logq(X)p(X)​​=n1​log∏q(xi​)∏p(xi​)​=n1​i=1∑n​logq(xi​)p(xi​)​⟶P​EX​logq(x)p(x)​​

这里最后一个收敛用的是 LLN，当 n 很大的时候，样本均值收敛到总体期望

当 H0H_0H0​ 成立时，随后一个 EEE 其实也就是 Ep(x)log⁡p/q=D(p∣∣q)E_{p(x)} \log p/q = D(p||q)Ep(x)​logp/q=D(p∣∣q)，则上述的对数似然比会依概率收敛到两个分布的 KL 散度，且随着 nnn 增大会集中在 D(p∣∣q)D(p||q)D(p∣∣q) 的很小的邻域内；似然比检验会依据我们容许的一类错误概率 ε\varepsilonε 划掉尾巴，尾巴之上的部分就接受 (似然比越大我们越倾向于 ppp，其实是单侧检验)；这里的接受域可以表示为：

accept if log-ratio = 1nlog⁡p(Xn)q(Xn)≥D(p∣∣q)−δεBn(ε)={(x1,...,xn):1nlog⁡p(Xn)q(Xn)≥D(p∣∣q)−δε}\text{accept if log-ratio = } \frac{1}{n} \log \frac {p(X^n)}{q(X^n)} \geq D(p||q) - \delta_\varepsilon \\ B_n^{(\varepsilon)} = \{(x_1, ..., x_n): \frac{1}{n} \log \frac {p(X^n)}{q(X^n)} \geq D(p||q) - \delta_\varepsilon\} accept if log-ratio = n1​logq(Xn)p(Xn)​≥D(p∣∣q)−δε​Bn(ε)​={(x1​,...,xn​):n1​logq(Xn)p(Xn)​≥D(p∣∣q)−δε​}

通过调整 δε\delta_\varepsilonδε​，我们可以控制第一类错误概率；假设这里要求一类错误不大于阈值 ε\varepsilonε，满足

P(Bn(ε))>1−εP(B_n^{(\varepsilon)}) > 1-\varepsilon P(Bn(ε)​)>1−ε

在这个基础上计算二类错误概率：

Q(Bn(ε))=Q(1nlog⁡p(X)q(X)≥D(p∣∣q)−δε)=Q[q(x)≤p(x)2−n(D−δε)]=∑xn: q(x)≤p(xn)2−n(D−δε)q(xn)≤∑xn: q(x)≤p(xn)2−n(D−δε)p(xn)2−n(D−δε)≤∑xnp(xn)2−n(D−δε)=2−n(D−δε)∑xnp(xn)≤2−n(D−δε)→2−nD(p∣∣q)\begin{aligned} Q(B_n^{(\varepsilon)}) &=Q\left(\frac{1}{n} \log \frac{p(X)}{q(X)} \geq D(p||q) - \delta_\varepsilon\right) \\ &= Q\left[q(x) \leq p(x) 2^{-n(D-\delta_\varepsilon)}\right] \\ &= \sum _{x^n: \,q(x) \leq p(x^n) 2^{-n(D-\delta_\varepsilon)}} q(x^n) \\ &\leq \sum _{x^n: \,q(x) \leq p(x^n) 2^{-n(D-\delta_\varepsilon)}} p(x^n) 2^{-n(D-\delta_\varepsilon)} \\ &\leq \sum_{x^n} p(x^n) 2^{-n(D-\delta_\varepsilon)} = 2^{-n(D-\delta_\varepsilon)}\sum_{x^n} p(x^n) \\ &\leq 2^{-n(D-\delta_\varepsilon)} \to 2^{-nD(p||q)} \end{aligned} Q(Bn(ε)​)​=Q(n1​logq(X)p(X)​≥D(p∣∣q)−δε​)=Q[q(x)≤p(x)2−n(D−δε​)]=xn:q(x)≤p(xn)2−n(D−δε​)∑​q(xn)≤xn:q(x)≤p(xn)2−n(D−δε​)∑​p(xn)2−n(D−δε​)≤xn∑​p(xn)2−n(D−δε​)=2−n(D−δε​)xn∑​p(xn)≤2−n(D−δε​)→2−nD(p∣∣q)​

综上，选用似然比作为接受 / 拒绝域的 threshold 时，二类错误概率的上界是 2−nD(p∣∣q)2^{-nD(p||q)}2−nD(p∣∣q)​ ；也就是此时的统计量收敛到自己的均值位置 D(p∣∣q)D(p||q)D(p∣∣q)，而二类错误概率依照两个需要区分的分布之间的距离 D(p∣∣q)D(p||q)D(p∣∣q) 的增大而 exponentially 减小；

继续证明，在控制一类错误概率 < ε\varepsilonε 的情况下，不可能有另一种拒绝域 / 接受域使得二类错误概率更低：

假设存在另一种接受域 BnB_nBn​, 使得 PPP 成立时 P(Bn)>1−αP(B_n) > 1-\alphaP(Bn​)>1−α ，也就是我们规定的一类错误概率；同时如果 Q 是对的，BnB_nBn​ 可以给出比上面的上界更高的检验功效：Q(Bn)≥2−nDQ(B_n) \geq 2^{-nD}Q(Bn​)≥2−nD

这里设定 AnA_nAn​ 是 D(p∣∣q)D(p||q)D(p∣∣q) 的典型集，也就是对于某一个 δn\delta_nδn​，它同时满足

P(An)≥1−δn1nlog⁡PQ∈[D−εn,D+εn]P(A_n) \geq 1 -\delta_n \\ \frac1n \log \frac{P}{Q} \in [D-\varepsilon_n, D+\varepsilon_n] P(An​)≥1−δn​n1​logQP​∈[D−εn​,D+εn​]

综上，满足

Q(Bn)≥Q(An∩Bn)=∑xn∈An∩BnQ(xn)≥∑xn∈An∩BnP(xn)2−n(D+εn)=2−n(D+εn)∑xn∈An∩BnP(xn)=2−n(D+εn)P(An∩Bn)≥2−n(D+εn)(1−α−δn)=2−n(D+εn−log⁡2(1−α−δn)/n)→2−n(D+ε′)\begin{aligned} Q(B_n) &\geq Q(A_n \cap B_n) \\ &= \sum_{x^n \in A_n \cap B_n} Q(x^n) \\ &\geq \sum _{x^n \in A_n \cap B_n} P(x^n) 2^{-n(D + \varepsilon_n)} \\ &= 2^{-n(D + \varepsilon_n)} \sum _{x^n \in A_n \cap B_n} P(x^n) \\ &= 2^{-n(D + \varepsilon_n)} P(A_n \cap B_n) \\ &\geq 2^{-n(D + \varepsilon_n)}(1-\alpha -\delta_n) \\ &= 2^{-n(D + \varepsilon_n - \log_2(1-\alpha-\delta_n)/n)} \\ &\to 2^{-n(D + \varepsilon')} \end{aligned} Q(Bn​)​≥Q(An​∩Bn​)=xn∈An​∩Bn​∑​Q(xn)≥xn∈An​∩Bn​∑​P(xn)2−n(D+εn​)=2−n(D+εn​)xn∈An​∩Bn​∑​P(xn)=2−n(D+εn​)P(An​∩Bn​)≥2−n(D+εn​)(1−α−δn​)=2−n(D+εn​−log2​(1−α−δn​)/n)→2−n(D+ε′)​

3.3 高概率集和典型集

前面提到了典型集满足两个性质：

  • Pr(Aε(n))→1Pr(A _\varepsilon ^{(n)}) \to 1Pr(Aε(n)​)→1
  • ∣Aε(n)∣≤2(nH(X)−ε)|A _\varepsilon ^{(n)}| \leq 2 ^{(nH(X)-\varepsilon)}∣Aε(n)​∣≤2(nH(X)−ε)

也就是说，典型集应该是一个元素个数有限，但是把最常出现的序列都包括全了的集合；它可以保证任意一个来自总体 p(x)p(x)p(x) 的新的序列 (x1,...,xn)(x_1, ..., x_n)(x1​,...,xn​) 以接近 1 的概率是典型集中的序列，也能保证内部序列的个数不是太多，所以用 3.2 中的数据压缩法 (先分典型集和非典型集，然后用 index 指代每一个序列) 的时候，典型集的 index 不至于太大，保证了编码平均下来最短；

那问题来了，我们 3.2 进行数据压缩编码的时候，index 的个数也就是典型集内元素的个数，也就是说在保证典型集覆盖 Pr→1Pr\to 1Pr→1 的基础上，典型集内的元素是越少越好的，越能降低我的编码长度；则我们考虑所有这些包括了绝大多数情况的，满足 Pr(Bε(n))→1Pr(B _\varepsilon ^{(n)}) \to 1Pr(Bε(n)​)→1 的集合中，有没有比典型集元素个数更少的？
