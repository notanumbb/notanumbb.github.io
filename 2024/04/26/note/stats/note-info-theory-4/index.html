信息论 (4): 随机过程序列的性质与熵率

发表于2024-04-26|statsinformation theory

笔记基于 Elements of Information Theory (second edition) - Thomas M. Cover, Joy A. Thomas；

其中部分含有个人理解，疏漏还请多指正。


本节讨论随机过程序列的熵的性质，给出熵率的定义；

  • 随机过程与马尔科夫性

  • 熵率 (entropy rate)

回忆：AEP 告诉我们当 n 足够大时，任一可能的序列 (x1,x2,...,xn)(x_1, x_2, ..., x_n)(x1​,x2​,...,xn​) 依概率收敛到一个和熵相关的常数值，并基于此构造典型集

那如果这里的序列的多个随机变量之间不是独立的呢？或者说，如果他们来自于某种平稳序列的情况呢？

可以证明，对于一个 stochastic sequence，此时的序列的熵 H(X1,X2,...,Xn)H(X_1, X_2, ..., X_n)H(X1​,X2​,...,Xn​) 近似为一个 nnn 的线性函数，以 H(X)H(\mathcal{X})H(X) 为增长率增长；也就是

H(X1,X2,...,Xn,Xn+1)−H(X1,X2,...,Xn)→H(X)H(X_1, X_2, ..., X_n, X_{n + 1}) - H(X_1, X_2, ..., X_n) \to H(\mathcal{X}) H(X1​,X2​,...,Xn​,Xn+1​)−H(X1​,X2​,...,Xn​)→H(X)

则此时我们将 H(X)H(\mathcal{X})H(X)​ 称作这个过程 (process) 的熵率 (entropy rate)


4.1 随机过程与马尔可夫性

首先理解到底什么是随机过程 (stochastic process)，简单理解，随机过程 {Xt,t∈T}\{X_t, t\in T\}{Xt​,t∈T} 也就是一系列的有编号 (indexed) 的，随机变量的序列 (sequence)，其中我们的 index 来自于指标集 TTT

平稳性 (stationary) ：若随机变量序列的联合分布并不随时间点 (time index) 变化，则我们称这个随机过程为平稳的；也就是对于 ∀t>0\forall t > 0∀t>0，下式成立

Pr(X1=x1,X2=x2,...,Xn=xn)=Pr(X1+t=x1,X2+t=x2,...,Xn+t=xn)Pr(X_1 = x_1, X_2 = x_2, ..., X_n = x_n) = \\ Pr(X_{1+t} = x_1, X_{2+t} = x_2, ..., X_{n+t} = x_n) Pr(X1​=x1​,X2​=x2​,...,Xn​=xn​)=Pr(X1+t​=x1​,X2+t​=x2​,...,Xn+t​=xn​)

说白了也就是任意的长度为 nnn 的序列的联合分布只和这里的长度 nnn 有关，和这个序列开始的时间点 ttt 无关；其实对于 n=1n=1n=1 的情况也就是常说的平稳性：Pr⁡(X1)=Pr⁡(X2)=...=Pr⁡(Xn)\Pr(X_1) = \Pr(X_2) = ... = \Pr(X_n)Pr(X1​)=Pr(X2​)=...=Pr(Xn​)​

马尔可夫性 (Markov property)：对于时间点 t 的随机变量 XtX_tXt​，满足

P(Xt∣X1,...,Xt−1)=P(Xt∣Xt−1)P(X_t | X_1, ..., X _{t-1}) = P(X_t | X _{t-1}) P(Xt​∣X1​,...,Xt−1​)=P(Xt​∣Xt−1​)

也就是说，虽然前面历史上有 t-1 个随机变量可用，但是其实只要前一个时间点的 Xt−1X_{t-1}Xt−1​ 就已经足够提供了所有有关 XtX_tXt​ 的分布的信息了

  • 换句话说， "只要有了 Xt−1X_{t-1}Xt−1​ 的信息，再往前的信息都是冗余"；XtX_tXt​ 和前面所有的 X1,...,Xt−2X_1, ..., X _{t-2}X1​,...,Xt−2​ 都是 (conditioned on Xt−1X_{t-1}Xt−1​​) 条件独立的；

  • 在马尔可夫性下，一个序列的联合分布就可以写成初始分布 * 一系列条件分布的形式：

    首先考虑条件分布的链式法则：

    P(X1,...,Xn)=P(X1)P(X2∣X1)P(X3∣X2,X1),...,P(Xn∣X1,...,Xn−1)P(X_1, ..., X_n) = P(X_1) P(X_2|X_1) P(X_3|X_2, X_1), ..., P(X_n|X_1, ..., X_{n-1}) P(X1​,...,Xn​)=P(X1​)P(X2​∣X1​)P(X3​∣X2​,X1​),...,P(Xn​∣X1​,...,Xn−1​)

    而每一个条件概率的 条件 都可以简化为最近的一条信息 (马尔可夫性)

    P(X1=x1,...,Xn=xn)=P(X1=x1)P(X2=x2∣X1=x1)...P(Xn=xn∣Xn−1=xn−1)P(X_1=x_1, ..., X_n= x_n) \\= P(X_1=x_1) P(X_2=x_2|X_1=x_1) ... P(X_n=x_n|X_{n-1}=x_{n-1}) P(X1​=x1​,...,Xn​=xn​)=P(X1​=x1​)P(X2​=x2​∣X1​=x1​)...P(Xn​=xn​∣Xn−1​=xn−1​)

也就是说，此时只要知道 1) 初始分布 ；2) 转移矩阵；我们就可以定义一个随机过程；

综上，定义一个随机过程为 马尔可夫链 Markov chain，需要满足：

  • 状态空间 state space = XtX_tXt​ 的取值范围是可数的 (countable)
  • $X_1, ..., X_t, ... $ 满足马尔可夫性 Markov property

另外，除非特别说明，我们关心的都是 时间不变 (time invariant / time homogeneous) 的马尔科夫链；即此时条件概率 p(Xn+1∣Xn)p(X_{n+1} | X_{n})p(Xn+1​∣Xn​) 和具体的时间点 nnn 无关，即

p(Xn+1∣Xn)=p(X2∣X1)p(X_{n+1} | X_n) = p(X_2 | X_1) p(Xn+1​∣Xn​)=p(X2​∣X1​)

类似地，马尔科夫链的状态转移都可以写成分布和转移矩阵相乘的形式：

πt+1=πtP\pi_{t+1} = \pi_t P πt+1​=πt​P

接下来我们主要会考察平稳马尔科夫链，也就是经过任意多步的转移 PnP^nPn ，分布都保持 π\piπ 不变；



4.2 熵率 (entropy rates)


定义

回忆熵的链式法则：

H(X1,...,Xn,Xn+1)=H(X1,...,Xn)+H(Xn+1∣X1,...,Xn)H(X_1, ..., X_n, X_{n+1}) = H(X_1, ..., X_n) + H(X_{n+1}|X_1, ..., X_n) H(X1​,...,Xn​,Xn+1​)=H(X1​,...,Xn​)+H(Xn+1​∣X1​,...,Xn​)

换句话说，如果当前已经收到了 nnn 个信号，得到一个 nnn 长度的序列，则再收到一个新的信号 Xn+1X_{n+1}Xn+1​ 时，我们的熵会对应增加 H(Xn+1∣X1,...,Xn)H(X_{n+1} | X_1, ..., X_n)H(Xn+1​∣X1​,...,Xn​)

对于 iidiidiid 的情况，此时条件熵也就等价于 H(X)H(X)H(X)，则每增加一个随机变量，序列的熵会增长常数 H(X)H(X)H(X)，这也就是 AEP 的结论；

考虑对于非 iidiidiid 的情况呢？我们将这里的序列建模为一个随机过程，并考虑过程长期的表现； n→∞n\to\inftyn→∞ 时，我们每一轮的增长量也就是条件熵的极限：

H′(X)=lim⁡n→∞H(Xn∣X1,...,Xn−1)H'(\mathcal{X}) = \lim _{n\to \infty} H(X_n|X_1, ..., X_{n-1}) H′(X)=n→∞lim​H(Xn​∣X1​,...,Xn−1​)

可以证明，对于平稳随机序列，条件熵 H(Xn∣X1,...,Xn−1)H(X_n | X_1, ..., X _{n-1})H(Xn​∣X1​,...,Xn−1​) 非递增 (non-increasing)，极限存在​

H(Xn+1∣X1,...,Xn)≤H(Xn∣X1,...,Xn)H(X _{n+1} | X_1, ..., X_n) \leq H(X_n | X_1, ..., X_n) H(Xn+1​∣X1​,...,Xn​)≤H(Xn​∣X1​,...,Xn​)

证明：

因为这里是平稳随机过程，所以给定长度为 nnn 的历史时的条件分布和起始时间点 ttt 无关；此时 Xn∣X1,...,Xn−1X_{n}|X_1, ..., X_{n-1}Xn​∣X1​,...,Xn−1​ 的条件分布，和 Xn+1∣X2,...,XnX_{n+1}|X_2, ..., X_nXn+1​∣X2​,...,Xn​ 的条件分布是一样的；

而条件熵就是条件分布的函数，所以它俩的条件熵也是一样的；然而，新加条件一定会导致熵的减少；

H(Xn∣X1,...,Xn−1)=H(Xn+1∣X2,...,Xn)≥H(Xn+1∣X1,X2,...,Xn)\begin{aligned} H(X_n | X_1, ..., X _{n-1}) &= H(X_{n+1} |X_2, ..., X_n) \\ &\geq H(X_{n+1}|X_1, X_2, ..., X_n) \end{aligned} H(Xn​∣X1​,...,Xn−1​)​=H(Xn+1​∣X2​,...,Xn​)≥H(Xn+1​∣X1​,X2​,...,Xn​)​

而熵又是非负的，一个非递增的非负序列一定是收敛的；

直观理解，对于平稳序列此时联合分布和时间点的移动无关，并不会存在一个特别的 time index 使得整个序列的分布发生什么大变；而熵就是分布的函数，则熵也不会大变；

进一步地，当 n→∞n\to \inftyn→∞ 时，我积累的历史信息越来越多，在给定历史下的 XnX _{n}Xn​​ 的条件熵应该是减少或至少保持不变的；

综上，这个条件熵会收敛到某一个值，表示那一部分 "即使积累再多的历史信息也无法降低的" 不确定性；

再进一步，根据熵的链式法则，我们有

H(X1,...,Xn)=∑i=1nH(Xi∣Xi−1,...,X1)1nH(X1,...,Xn)=1n∑i=1nH(Xi∣Xi−1,...,X1)H(X_1, ..., X_n) = \sum _{i=1}^n H(X_i | X _{i-1}, ..., X_1) \\ \frac{1}{n} H(X_1, ..., X_n) = \frac{1}{n} \sum _{i=1}^n H(X_i | X _{i-1}, ..., X_1) H(X1​,...,Xn​)=i=1∑n​H(Xi​∣Xi−1​,...,X1​)n1​H(X1​,...,Xn​)=n1​i=1∑n​H(Xi​∣Xi−1​,...,X1​)

证明：

我们已知右边级数的数列是收敛的 (H(Xn∣X1,...,Xn−1)H(X_n|X_1, ..., X_{n-1})H(Xn​∣X1​,...,Xn−1​)​ 收敛)；

则对于一个收敛数列，它的级数的平均值也是收敛的，且二者的极限相同

Note

这里用到了 Cesaro mean ；

正式的表示为：若序列 bn→bb_n \to bbn​→b 收敛，则 bnb_nbn​ 的前 nnn 项的平均值 1/n∑bi1/n \sum b_i1/n∑bi​ 也随着 n→∞n\to \inftyn→∞ 收敛到 bbb

bn→b  ⟹  1n∑i=1nbi→bb_n \to b \quad\implies \quad \frac{1}{n} \sum _{i=1}^n b_i \to b bn​→b⟹n1​i=1∑n​bi​→b

直观理解，当 n 足够大的时候，前面离 b 很远的那些 bnb_nbn​ 只有有限项，会被后面的超多的离 b 很近的 bnb_nbn​​​ 稀释，最后的平均值也是收敛到 b 的

综上，对于一个平稳序列，此时下面两个极限都存在，且相等

H(X)=lim⁡1nH(X1,...,Xn)=lim⁡1n∑nH(Xi∣X1,...,Xi−1)H(\mathcal{X}) = \lim \frac{1}{n} H(X_1, ..., X_n) = \lim \frac{1}{n} \sum^n H(X_i|X_1, ..., X _{i-1}) H(X)=limn1​H(X1​,...,Xn​)=limn1​∑n​H(Xi​∣X1​,...,Xi−1​)

而这里的 H(X)H(\mathcal{X})H(X) 也就表示了当 n→∞n\to\inftyn→∞​ 时，序列每增加一个单位带来的熵的增长量，我们把它记作熵率

给出严谨的定义：

考虑 平稳随机过程 (stationary stochastic process)，下式两个极限存在且相等，记作 **熵率 (entropy rates) **，用 H(X)H(\mathcal{X})H(X) 表示；表示序列每增加一个单位，联合熵对应增加常数 H(X)H(\mathcal{X})H(X)

H(X)=lim⁡n→∞H(Xn∣X1,...,Xn−1)=lim⁡n→∞1nH(X1,...,Xn)\begin{aligned} H(\mathcal{X}) &= \lim _{n\to \infty} H(X_n|X_1, ..., X_{n-1}) \\ &= \lim _{n\to \infty} \frac1n H(X_1, ..., X_n) \end{aligned} H(X)​=n→∞lim​H(Xn​∣X1​,...,Xn−1​)=n→∞lim​n1​H(X1​,...,Xn​)​

Important

总结：

  • 平稳随机过程的熵率也就是随着序列不断加长，整个序列的联合熵的增加量；每新产生一个随机变量，序列增长一个单位，则大家的联合熵增加一个常数 H(X)H(\mathcal{X})H(X)​
  • 根据 Cesaro mean，由于单位增长率收敛，则此时整个序列的均值 = 1n\frac1nn1​​联合熵 也就收敛到熵率；
  • 类比 AEP：对于 iidiidiid 的序列，增长率 H(Xn∣X1,...,Xn−1)H(X_n|X_1, ..., X_{n-1})H(Xn​∣X1​,...,Xn−1​) 是 exactly 等于 H(X)H(X)H(X) 的；但是对于非 iidiidiid 的情况，只要序列平稳，则它在长期的时候增长率也会收敛到一个常数 H(X)H(\mathcal{X})H(X)；可以说 AEP 也就是特殊情况下的熵率的性质；

常见随机过程的熵率

  • 每一个元素 XnX_nXn​ 都是独立的 uniformly 的分布，其中 ∣X∣=m|\mathcal{X}| = m∣X∣=m

    则联合分布 P(x1,...,xn)=1/mnP(x_1, ..., x_n) = 1/m^nP(x1​,...,xn​)=1/mn，对应

    H(x1,...,xn)=∑mnnmnlog⁡m=nlog⁡mH(X)=lim⁡n→∞nnlog⁡m=log⁡mH(x_1, ..., x_n) = \sum ^{m^n}\frac{n}{m^n} \log m = n\log m\\ H(\mathcal{X}) = \lim _{n\to \infty} \frac{n}{n} \log m = \log m H(x1​,...,xn​)=∑mn​mnn​logm=nlogmH(X)=n→∞lim​nn​logm=logm

  • 推广，对于 i.i.di.i.di.i.d 的随机变量，此时的随机过程的熵

    H(X1,...,Xn)=nH(X)H(X)=lim⁡n→∞n/nH(X)=H(X)H(X_1,..., X_n)= n H(X) \\ H(\mathcal{X}) = \lim _{n\to \infty } n/n H(X) = H(X) H(X1​,...,Xn​)=nH(X)H(X)=n→∞lim​n/nH(X)=H(X)

  • 如果独立但是不同分布呢

    H(X1,...,Xn)=∑i=1nH(Xi)H(X)=lim⁡n→∞1n∑i=1n(Xi)H(X_1,..., X_n)= \sum _{i=1}^n H(X_i) \\ H(\mathcal{X}) = \lim _{n\to \infty} \frac{1}{n} \sum _{i=1}^n(X_i) H(X1​,...,Xn​)=i=1∑n​H(Xi​)H(X)=n→∞lim​n1​i=1∑n​(Xi​)

    也就是共 n 个元素的熵的平均；这里的极限就不一定存在了；
