13.1 经验分布 (type) 与 type class

定义

定义 probability simplex：即一系列长度为 mmm 的向量 (x1,...,xm)(x_1, ..., x_m)(x1​,...,xm​)，满足 xi≥0, ∑i=1mxi=1x_i \geq 0, \, \sum_{i=1}^m x_i = 1xi​≥0,∑i=1m​xi​=1；

注意到这里有加和为 1 的约束，所以实际上只有 m-1 维的自由度；

对应一个维度的 probability simplex 向量的全体会组成一个平面；比如 n = 3 时：

考虑取值范围为 X∈XX \in \mathcal{X}X∈X，满足分布为 QQQ，此时存在 nnn 个 iid 的样本 x1,...,xnx_1, ..., x_nx1​,...,xn​​

    P^(xj)=Nxj/n\hat P(x_j) = N_{x_j} / n P^(xj​)=Nxj​​/n

  • 此时原始分布的概率 Pr⁡(X=j)=qj\Pr(X = j) = q_jPr(X=j)=qj​ ；则这个长度为 nnn 的序列对应的联合分布

    q(x1,...,xn)=∏i=1nq(xi)=∏x∈Xq(x)Nx=exp⁡{∑x∈XNxlog⁡q(x)}=exp⁡{n∑x∈X[(Nx/n)log⁡(Nx/n)−(Nx/n)log⁡Nx/nq(x)]}=exp⁡{−n(H(p)+D(p∣∣q))}\begin{aligned} q(x_1, ..., x_n) &= \prod_{i=1}^n q(x_i) \\ &= \prod _{x\in \mathcal{X}} q(x)^{N_{x}} \\ &= \exp \{\sum_{x\in \mathcal{X}} N_x\log q(x)\} \\ &= \exp \left\{n\sum_{x\in \mathcal{X}} \left[(N_x/n)\log (N_x/n) - (N_x/n)\log \frac{N_x/n}{q(x)}\right]\right\} \\ &= \exp \{-n (H(p) + D(p || q))\} \end{aligned} q(x1​,...,xn​)​=i=1∏n​q(xi​)=x∈X∏​q(x)Nx​=exp{x∈X∑​Nx​logq(x)}=exp{nx∈X∑​[(Nx​/n)log(Nx​/n)−(Nx​/n)logq(x)Nx​/n​]}=exp{−n(H(p)+D(p∣∣q))}​

对于每一个 xnx^nxn 序列，出现的联合分布只和原始的分布 qqq 和这个序列的 type ppp​ 有关，和具体的每一个元素的出现顺序无关；

综上，只要多个序列对应相同的 type ，也就是每一个可能的元素出现次数相同 (即使出现顺序不同)；从本质上来说它们的联合分布就相同；

T(p)={(x1,...,xn):p^(x)=Nx/n=p(x)}T(p) = \{(x_1, ..., x_n): \hat p(x) = N_x/n = p(x)\} T(p)={(x1​,...,xn​):p^​(x)=Nx​/n=p(x)}

13.2 大数定律 (LLN)

考虑当前想要用 type 来估计原来的分布 Q，用 KL 散度来衡量我们这里估计的误差

则可以定义这些 (对应的 type 和原分布 QQQ 差不远) 的 n 维序列组成的 typical set：

TQε={Xn=(x1,...,xn);D(PXn∣∣Q)<ε}T_Q^\varepsilon = \{X^n = (x_1, ..., x_n); D(P_{X^n} || Q) < \varepsilon\} TQε​={Xn=(x1​,...,xn​);D(PXn​∣∣Q)<ε}

则 typical set 以外的概率满足：

1−Qn(TQε)=∑D(p∣∣Q)>εQ(T(p))≤∑D(p∣∣Q)>εe−n(D(p∣∣q))≤∑D(p∣∣Q)>εe−nε≤(n+1)∣X∣e−nε=exp⁡{−n[ε−∣X∣ln⁡(n+1)/n]}\begin{aligned} 1-Q^n(T_Q^\varepsilon) &= \sum _{D(p||Q) > \varepsilon} Q(T(p)) \\ &\leq \sum _{D(p||Q) > \varepsilon} e^{-n(D(p||q))} \\ &\leq \sum _{D(p||Q) > \varepsilon} e^{-n \varepsilon} \\ &\leq (n + 1)^{|\mathcal{X}|} e^{-n \varepsilon} \\ &= \exp \{-n [\varepsilon - |\mathcal{X}| \ln (n+1) /n]\} \\ \end{aligned} 1−Qn(TQε​)​=D(p∣∣Q)>ε∑​Q(T(p))≤D(p∣∣Q)>ε∑​e−n(D(p∣∣q))≤D(p∣∣Q)>ε∑​e−nε≤(n+1)∣X∣e−nε=exp{−n[ε−∣X∣ln(n+1)/n]}​

当 n→∞n \to \inftyn→∞ 的时候，ln⁡(n+1)/n→0\ln (n + 1)/n \to 0ln(n+1)/n→0， 上式其实就是收敛到 exp⁡{−nε}→0\exp \{-n \varepsilon\} \to 0exp{−nε}→0​，也就是说出现那种 (对应的 type 离真实分布很远) 的 n 维序列的概率是趋向 0 的

Pr⁡{D(PXn∣∣Q)>ε}→0,n→∞\Pr \{D(P_{X^n} || Q) > \varepsilon\} \to 0, \quad n \to \infty Pr{D(PXn​∣∣Q)>ε}→0,n→∞

换个角度，考虑随着 n→∞n \to \inftyn→∞ 此时分布 $\hat p $ 依概率收敛到 QQQ，计算

令 PnP_nPn​ 表示这里的所有的 n 维 type 的全集，而这里的 EEE 是 n 维分布的任意一个子集，里面有一堆 n 维的分布；则一个 n 维样本对应的 type 出现在 EEE​ 的概率可以表示为：

13.4 conditional limit theorem

pythagorean theorem

引理：相对熵的三角不等式；

假设这里的 P∗P^*P∗ 是 E 中离 QQQ 真实分布最近的一个分布，则对于任意的 P∈EP \in EP∈E ，满足：

D(P∣∣Q)≥D(P∣∣P∗)+D(P∗∣∣Q)D(P||Q) \geq D(P||P^*) + D(P^*||Q) D(P∣∣Q)≥D(P∣∣P∗)+D(P∗∣∣Q)

如果此时 P∗P^*P∗ 为指数分布族形式 (形如 ch12 对应的最大熵模型 ∝q(x)eλ+∑λiri(x)\propto q(x) e^{\lambda + \sum \lambda_i r_i(x)}∝q(x)eλ+∑λi​ri​(x))，则等式成立：

D(P∣∣Q)−D(P∗∣∣Q)=D(P∣∣P∗)D(P || Q) - D(P^*||Q) = D(P||P^*) D(P∣∣Q)−D(P∗∣∣Q)=D(P∣∣P∗)

试着从几何上理解：

考虑这里的集合 E，想要找到 EEE 中离我们的目标 QQQ 最近的一个点，实际上就是要求这个 P∗P^*P∗ 满足 (P* 位置关于 E 的切平面) 和 (PQ 连线) 垂直；综上，对于任意的另一个点 PPP​，这里的角实际上是一个直/钝角；

实际上和欧氏距离也是类似的；考虑存在一个凸集 (convex set，回忆相对熵是凸函数)，假设离外部的点 AAA 最近的集合内的元素为 BBB，同时存在任意的集合内部的点 CCC，则欧氏距离满足

lAC2≥lAB2+lBC2l^2 _{AC} \geq l^2_{AB} + l^2_{BC} lAC2​≥lAB2​+lBC2​

另一个应用在于：

如果此时存在另一个 PPP 使得 D(P∣∣Q)D(P||Q)D(P∣∣Q) 接近于 E→QE \to QE→Q 的最小距离 D(P∗∣∣Q)D(P^*||Q)D(P∗∣∣Q) ，则可以证明这里的 PPP 离我们的 $P^* $ 也很近，满足 D(P∣∣P∗)→0D(P||P^*) \to 0D(P∣∣P∗)→0​

也就是说，如果 D(P∗∣∣Q)D(P^*||Q)D(P∗∣∣Q) 是我们能够达到的，EEE 和目标分布 QQQ 最近的距离，则可以证明这些离 QQQ 最近的 EEE 内的点实际上全是 "抱团" 的，彼此之间也不会离得太远；不会出现那种独立一群离 QQQ 很近，另一群也离 QQQ 很近的情况；

定义另一种两个分布之间的距离：

∣∣P1−P2∣∣1=∑a∈X∣P1(a)−P2(a)∣||P_1 - P_2||_1 = \sum_{a \in \mathcal{X}} |P_1(a) - P_2(a)| ∣∣P1​−P2​∣∣1​=a∈X∑​∣P1​(a)−P2​(a)∣

∣∣P1−P2∣∣1=2(P1(A)−P2(A));A={a:P1(a)>P2(a)}||P_1 - P_2||_1 = 2(P_1(A) - P_2(A)); \quad A = \{a: P_1(a) > P_2(a)\} ∣∣P1​−P2​∣∣1​=2(P1​(A)−P2​(A));A={a:P1​(a)>P2​(a)}

再进一步可以证明相对熵和它存在不等式关系：

D(P1∣∣P2)≥12ln⁡2∣∣P1−P2∣∣12D(P_1||P_2) \geq \frac {1}{2\ln 2}||P_1 - P_2||^2_1 D(P1​∣∣P2​)≥2ln21​∣∣P1​−P2​∣∣12​


conditional limit theorem

综上，可以得到 conditional limit theorem：

规定 Xn=(x1,...,xn)X^n = (x_1, ..., x_n)Xn=(x1​,...,xn​) 为 iid 的来自于分布 QQQ 的随机变量，对应 type 为 PXnP_{X^n}PXn​；当给定条件：PXn∈EP_{X^n} \in EPXn​∈E 时，规定 EEE 中使得 D(P∗∣∣Q)D(P^*||Q)D(P∗∣∣Q) 最小的分布为 P∗P^*P∗，则此时条件概率满足：

Pr⁡(X1=a∣PXn∈E)→P∗(a);P∗=arg⁡min⁡P∈ED(P∣∣Q)\Pr(X_1 = a|P_{X^n} \in E) \to P^*(a); \quad P^* = \arg \min _{P\in E} D(P||Q) Pr(X1​=a∣PXn​∈E)→P∗(a);P∗=argP∈Emin​D(P∣∣Q)

换言之，虽然真实的 XXX 的分布来自于 QQQ，但是给定了 EEE 的条件后，此时的条件概率只和 P∗=arg⁡min⁡D(P∣∣Q)P^* = \arg \min D(P||Q)P∗=argminD(P∣∣Q)​ 有关；

引理：同时可以证明，此时的多个 XiX_iXi​ 之间在给定条件 PXn∈EP_{X^n} \in EPXn​∈E 下的条件概率还是近似独立的；

证明：

考虑 probability simplex 上的一个 convex set CCC，而 BBB 是所有 type 在 CCC 内的 n 维度向量的集合

Pr⁡Q(P^n∈C)=Q(B)=exp⁡{−D(QX1,...,Xn∣Pn∈C∣∣QX1,...,Xn)}D(QX1,...,Xn∣B∣∣QX1,...,Xn)=EXn∈Blog⁡Q(Xn∣B)Q(Xn)=log⁡1Q(B)\Pr_Q(\hat P_n \in C) = Q(B) = \exp\{-D(Q_{X_1, ..., X_n|P_n\in C} ||Q_{X_1, ..., X_n})\}\\ D(Q_{X_1, ..., X_n|B} ||Q_{X_1, ..., X_n}) = E_{X^n \in B} \log \frac{Q(X^n|B)}{Q(X^n)} = \log \frac 1{Q(B)} QPr​(P^n​∈C)=Q(B)=exp{−D(QX1​,...,Xn​∣Pn​∈C​∣∣QX1​,...,Xn​​)}D(QX1​,...,Xn​∣B​∣∣QX1​,...,Xn​​)=EXn∈B​logQ(Xn)Q(Xn∣B)​=logQ(B)1​

换言之如果这里的 n 个 xix_ixi​ 是 iid 的，此时

D(QXn∣B∣∣QXn)=D(QXn∣B∣∣QX1∣B...QXn∣B)+∑i=1nD(QXn∣B∣∣QXi)D(Q_{X^n|B}||Q_{X^n}) = D(Q_{X^n|B} || Q_{X_1|B} ... Q_{X_n|B}) + \sum _{i=1}^n D(Q_{X^n|B} || Q_{X_i}) D(QXn∣B​∣∣QXn​)=D(QXn∣B​∣∣QX1​∣B​...QXn​∣B​)+i=1∑n​D(QXn∣B​∣∣QXi​​)

则上述的 Q(B)Q(B)Q(B) 可以继续表示为

Q(B)=exp⁡{−D(QX1,...,Xn∣Pn∈C∣∣QX1,...,Xn)}≤exp⁡{−n1n∑i=1nD(PXi∣B∣∣Q)}≤exp⁡{−nD(1n∑i=1nPXi∣B∣∣Q}≤exp⁡{−n[D(P∗∣∣Q)+D(PXi∣B∣∣P∗)]}\begin{aligned} Q(B) &= \exp\{-D(Q_{X_1, ..., X_n|P_n\in C} ||Q_{X_1, ..., X_n})\}\\ &\leq \exp\{-n \frac 1n \sum _{i=1}^nD(P_{X_i|B}||Q)\} \leq \exp \{-n D(\frac 1n\sum _{i=1}^n P_{X_i|B} ||Q\} \\ &\leq \exp\{-n[D(P^*||Q) + D(P_{X_i|B}||P^*)]\} \end{aligned} Q(B)​=exp{−D(QX1​,...,Xn​∣Pn​∈C​∣∣QX1​,...,Xn​​)}≤exp{−nn1​i=1∑n​D(PXi​∣B​∣∣Q)}≤exp{−nD(n1​i=1∑n​PXi​∣B​∣∣Q}≤exp{−n[D(P∗∣∣Q)+D(PXi​∣B​∣∣P∗)]}​

也就是给定了 BBB 时多个 XiX_iXi​ 还是近似独立的；



13.5 hypothesis test

考虑存在 n 个 iid 来自于 QQQ 的样本 Xn=x1,...,xnX^n = x_1, ..., x_nXn=x1​,...,xn​，进行假设检验：

H0=P1;   H1=P2H_0 = P_1; \,\,\, H_1 = P_2 H0​=P1​;H1​=P2​

定义两种 error：

  • 一类错误 α\alphaα，也就是 H0H_0H0​ 正确但是反而拒绝了 H0H_0H0​ 的概率：

    α=P1(reject H0)\alpha = P_1(\text{reject }H_0) α=P1​(reject H0​)

  • 二类错误 β\betaβ ，也就是 H1H_1H1​ 正确但反而接受了 H0H_0H0​ 的概率；同时定义 1−β1-\beta1−β 为检验的功效 = power，也就是 H1H_1H1​ 正确且拒绝了 H0H_0H0​ = 接受了 H1H_1H1​ 的概率；

    β=P2(accept H0)power=1−β=P2(reject H0)\beta = P_2(\text{accept }H_0) \\ \text{power} = 1-\beta = P_2 (\text{reject } H_0) β=P2​(accept H0​)power=1−β=P2​(reject H0​)

常用的检验方法为似然比检验，也就是计算统计量 = 似然比，当比值超过某一个阈值时，考虑接受 H0H_0H0​

定义这里的阈值为 TTT，则接受域为 An(T)A_n(T)An​(T)：

An(T)={(x1,...,xn): P(x1,...,xn)P2(x1,...,xn)>T}A_n(T) = \left\{(x_1, ..., x_n) : \, \frac{P(x_1, ..., x_n)}{P_2(x_1, ..., x_n)} > T\right\} An​(T)={(x1​,...,xn​):P2​(x1​,...,xn​)P(x1​,...,xn​)​>T}

在接受域 An(T)A_n(T)An​(T)​ 下，一二类错误率可以表示为

α∗=P1(Anc(T));β=P2(An(T));power=1−β=P2(Anc(T))\alpha^* = P_1(A^c_n(T)); \quad \beta = P_2(A_n(T)); \quad \text{power} = 1-\beta = P_2(A_n^c(T)) α∗=P1​(Anc​(T));β=P2​(An​(T));power=1−β=P2​(Anc​(T))

这里的错误率可以通过调整 TTT 来控制；


Neyman–Pearson lemma

对于如上定义的似然比检验的接受域 AnA_nAn​，和任意的其他检验的任意接受域 BnB_nBn​，令 AnA_nAn​ 对应的错误率为 α∗,β∗\alpha^*, \beta^*α∗,β∗，而 BnB_nBn​ 对应的错误率为 α,β\alpha, \betaα,β；

可以证明，此时若满足 α≤α∗\alpha \leq \alpha^*α≤α∗，则一定有 β≥β∗\beta \geq \beta^*β≥β∗

换言之，在规定了一类错误不大于某一个阈值的情况下，可以证明如上定义的似然比检验对应的二类错误是最低的；

chernoff-stein lemma：考虑如上定义的假设检验，对于任意的接受域 An(T)A_n(T)An​(T)，对应的一二类错误率为

13.6 fisher information & CR inequality

definition

考虑此时我们想要通过样本估计原始分布中的一个未知参数 θ∈Θ\theta \in \Thetaθ∈Θ；

规定这里对应了不同的 θ\thetaθ 的原始分布为 f(x;θ)f(x; \theta)f(x;θ)；我们通过一个函数 TTT 来进行估计，则此时的估计量为

θ^=T(x1,...,xn)\hat \theta = T(x_1, ..., x_n) θ^=T(x1​,...,xn​)

规定：

  • 满足 E[T(xn)]=θE[T(x^n)] = \thetaE[T(xn)]=θ 时，称无偏估计量 (unbiased estimator)

  • 满足 T(xn)⟶Pθ,  n→∞T(x^n ) \stackrel{P}{\longrightarrow} \theta, \, \, n \to \inftyT(xn)⟶P​θ,n→∞​ 时，称一致统计量 (consistent estimator)

  • 定义对数分布的，对未知参数的一阶导，为分布对应的 score，表示为

    V=∂∂θln⁡f(X;θ)V = \frac{\partial}{\partial \theta} \ln f(X; \theta) V=∂θ∂​lnf(X;θ)

    这里的 VVV 也就是一个 θ\thetaθ​ 相关的函数，考虑这里的 θ\thetaθ 是一个随机变量 (贝叶斯的思路，假设 θ\thetaθ 是有自己的分布的)，对应的 VVV 也就是一个随机变量

I(θ)=Eθ[V2]=Eθ[∂∂θln⁡f(X;θ)]2\begin{aligned} I(\theta) &= E_\theta [V^2] = E_\theta \left[\frac{\partial}{\partial \theta} \ln f(X; \theta)\right]^2 \end{aligned} I(θ)​=Eθ​[V2]=Eθ​[∂θ∂​lnf(X;θ)]2​

令 T(Xn)T(X^n)T(Xn) 为未知参数 θ\thetaθ 的 unbiased 无偏估计，则这个估计量的 mean-squared error = 估计量的方差
