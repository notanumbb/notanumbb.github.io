信息论 (6): 数据压缩与香农码

发表于2024-04-26|statsinformation theory

笔记基于 Elements of Information Theory (second edition) - Thomas M. Cover, Joy A. Thomas；

其中部分含有个人理解，疏漏还请多指正。


本节介绍信息论在数据压缩 / 编码原理的应用；

本质也就是找到合适的编码方式，对于任意的随机变量，用更短的 coding 表示更常见的数据，而用更长的 coding 表示不那么常出现的对象，使得整体的期望长度更短；

  • 编码的基本定义
  • Kraft 不等式
  • 最优编码界限与 Shannon coding


6.1 定义

定义：编码 (codes)，也就是将一个随机变量的取值范围做 mapping，实现 X→D∗\mathcal{X} \to D^*X→D∗

  • C(X)C(X)C(X)：表示随机变量 XXX 的编码
  • l(x)l(x)l(x)：表示原取值 xxx​ 对应的编码长度
  • C(x1x2...xn)=C(x1)C(x2)...C(xn)C(x_1x_2...x_n) = C(x_1)C(x_2)...C(x_n)C(x1​x2​...xn​)=C(x1​)C(x2​)...C(xn​)：也就是对于一串序列 x1,...,xnx_1, ..., x_nx1​,...,xn​​​，则它的编码就是每一个位置的元素对应编码然后再拼起来成一长串，记作编码的 extension
  • DDD：alphabet，也就是编码的基本组成单元的集合；比如二进制编码，则我们可用的最小的单元也就是 {0,1}\{0, 1\}{0,1}；而对于 DDD 进制，可用的最小单元也就是 {0,1,...,D−1}\{0, 1, ..., D-1\}{0,1,...,D−1}
  • D∗D^*D∗：用来表示 X\mathcal{X}X​ 的各个 coding 的集合；

Tip

举个例子，当前的随机变量 XXX 满足：

  • X={1,2,3}\mathcal{X} = \{1, 2, 3\}X={1,2,3}​，随机变量的取值范围；
  • D={0,1}D = \{0, 1\}D={0,1}，采用二进制编码；
  • C(1)=00,C(2)=10,C(3)=11C(1) = 00, C(2) = 10, C(3) = 11C(1)=00,C(2)=10,C(3)=11，编码方式，对每一个 X\mathcal{X}X 里可能的取值都有一个方案
  • D∗={00,10,11}D^* = \{00, 10, 11\}D∗={00,10,11}，实际上 coding 也就是实现 X→D∗\mathcal{X} \to D^*X→D∗

对于一种可能的编码方式，我们主要关心它的 平均编码长度 (expected length) L(C)L(C)L(C)

L(C)=∑x∈Xl(x)p(x)L(C) = \sum_{x\in \mathcal{X}} l(x)p(x) L(C)=x∈X∑​l(x)p(x)

  • 基本思想：用更短的 coding 表示更常见的数据，而用更长的 coding 表示不那么常出现的对象，使得整体的期望长度更短；

类似地，我们可以对编码做分类：

  • nonsingular：每一个 xxx 都对应唯一的 C(x)C(x)C(x)，不能不同的 xxx 被 mapping 成同一个 coding；

  • uniquely decodable：对于任意的序列 x1,...,xnx_1, ..., x_nx1​,...,xn​，此时 C(x1...xn)=C(x1)...C(xn)C(x_1...x_n) = C(x_1)...C(x_n)C(x1​...xn​)=C(x1​)...C(xn​)​ 也是满足 nonsingular 的

    举个例子，C(a) = 1, C(b) = 0, C(c) = 01，那出现 01 的时候，就会不知道到底是 ab 还是 c；

  • prefix code / instantaneous code：也就是任意的 x0x_0x0​ 的编码都不是其他任意 x1x_1x1​​​ 的编码的前缀；对于 prefix code，每一个编码只要一结束马上就能被认出来 (因为不是前缀)，所以能够保证是 uniquely decodable 的；

    但是注意 uniquely decodable 不一定要 prefix，比如编码 1 和 10，因为单独的 0 不存在，所以无论怎么组合还是能够解码分开 1 和 10

上述的几种分类存在包含关系：nonsingular →\to→ uniquely decodable →\to→ prefix code



6.2 kraft 不等式


Kraft 不等式

首先关注 prefix codes，可以证明，对于这样 prefix-free 的 coding 方式是存在限制的：

Kraft inequity：对于 DDD 进制的编码表 (alphabet，也就是共 {0,1,...,D−1}\{0, 1, ..., D-1\}{0,1,...,D−1} 这 DDD 个最小元素可用)，考虑随机变量 XXX，则每一个可能的取值 xxx 的即时码 (prefix code) 对应的编码长度 l(x)l(x)l(x) 必须对应满足：

∑x∈XD−l(x)≤1\sum_{x\in \mathcal{X}} D^{-l(x)} \leq 1 x∈X∑​D−l(x)≤1

  • 反过来说，如果此时存在一个随机变量和编码长度 l(x)l(x)l(x) 满足上面的式子，则一定存在一个 DDD​ 大小的 alphabet 对应的 prefix codes 编码方式；
  • 这里的 XXX 的取值范围 X\mathcal{X}X 也可以是无限集合；

证明：

因为是 prefix 编码，我们考虑用 DDD 叉树来表示；

想象一棵满 D 叉树 (也就是任意一个结点要么是叶子节点，要么有足足的 D 个子节点)；假设从根节点出发，每一次都以 1/D1/D1/D 的概率任选一个 child 向下，则最后落在一个第 l(x)l(x)l(x) 层的叶子节点的概率为 (1/D)l(x)(1/D)^{l(x)}(1/D)l(x)​

落在所有的叶子节点的概率的总和为 1，满足

∑x∈SD−layer(x)=1\sum _{x \in S}D^{-layer(x)} = 1 x∈S∑​D−layer(x)=1

对于任意的 D alphabet 的编码，我们都可以用一个 DDD 叉树上的叶子节点表示；考虑从 root 出发到某叶子节点的路径，如果在第 iii 层走到的第 jjj 个结点，就可以记这个编码的第 iii 位是 alphabet 中的第 jjj 个字符，如此这里的编码长度 l(x)l(x)l(x)​ 也就是这条路径最后的叶子节点的深度

考虑到编码的集合也就是叶子结点的集合的子集，满足

∑xD−l(x)≤1\sum _{x} D^{-l(x)} \leq 1 x∑​D−l(x)≤1

得证；

如果这里的编码对应一棵满 D 叉树的叶子节点，等式成立；换言之，如果这里的编码并没有把所有的叶子节点都用上，也就是说编码对应的树并非满二叉树，而存在一些没 assign 编码的叶子节点 / 一些子树不足 D 的中间结点 (也就是 redundant)，都会导致 Kraft 不等式不为 1


McMillan 不等式

类似 prefix code，所有的 uniquely decodable code 也要满足 kraft 不等式的形式：

McMillan inequity (Kraft in uniquely decodable code)：对于 DDD 进制的编码表 (alphabet，也就是共 {0,1,...,D−1}\{0, 1, ..., D-1\}{0,1,...,D−1} 这 DDD 个最小元素可用)，考虑随机变量 XXX，则每一个可能的取值 xxx 的唯一可解码 (uniquely decodable code) 对应的编码长度 l(x)l(x)l(x) 必须对应满足：

∑x∈XD−l(x)≤1\sum_{x\in \mathcal{X}} D^{-l(x)} \leq 1 x∈X∑​D−l(x)≤1

  • 反过来说，如果此时存在一个随机变量和编码长度 l(x)l(x)l(x) 满足上面的式子，则一定存在一个 DDD​ 大小的 alphabet 对应的 uniquely decodable codes 编码方式；
  • 这里的 XXX 的取值范围 X\mathcal{X}X​ 也可以是无限集合；

证明

考虑一个简单的证明：计算这里的 Kraft sum 的 k 次幂，如果这里的次幂没有爆炸，说明 Kraft sum 也应该是小于等于 1 的；

(∑D−l(x))k=∑x1∈X...∑xk∈XD−l(x1)...D−l(xk)=∑x1∈X...∑xk∈XD∑i=1k−l(xi)=∑(x1,...,xk)∈XkD−l(x1,...,xk)\begin{aligned} (\sum D^{-l(x)})^k &= \sum _{x_1\in \mathcal{X}} ... \sum _{x_k\in \mathcal{X}} D^{-l(x_1)} ... 也就是这里的 k 次幂也就等价于，所有可能的长度为 k 的序列对应的 Kraft sum

考虑这里的最长的编码 lmax(x)l_{max}(x)lmax​(x)，则这里的长度为 k 的序列的编码最长为 klmax(x)k l_{max}(x)klmax​(x)；假设这里的长度为 k 的序列的编码长度为 LLL ，则 L∈[1,klmax(x)]L \in [1, kl_{max}(x)]L∈[1,klmax​(x)]，且因为 uniquely，最多有 DLD^LDL 个长度为 k 的序列对应的 coding 可以是长度为 LLL 的

综上，继续上面的式子

(∑D−l(x))k≤∑L=1klmax(x)DLD−L=klmax(x)\begin{aligned} (\sum D^{-l(x)})^k &\leq \sum _{L=1}^{kl_{max}(x)} D^L D^{-L} = k l_{max}(x) \end{aligned} (∑D−l(x))k​≤L=1∑klmax​(x)​DLD−L=klmax​(x)​

换言之，这里的 Kraft sum 的 k 次幂是至多关于 kkk 线性的 (O(k)O(k)O(k))；而若这里的 ∑D−l(x)\sum D^{-l(x)}∑D−l(x) 比 1 大，对应的 k 次幂应该是指数级；

综上，∑D−l(x)≤1\sum D^{-l(x)} \leq 1∑D−l(x)≤1

McMillan 不等式也就是 Kraft 不等式从 prefix code 到 uniquely decodable codes 的延申；从此对编码的讨论可以从严格的 prefix code 延展到任意的 uniquely decodable codes

综上，对于一个随机变量 XXX，此时若能找到一组 l(x)l(x)l(x) 满足 ∑xD−l(x)≤1\sum_x D^{-l(x)} \leq 1∑x​D−l(x)≤1，则一定已找到一组对应长度的 uniquely decodable code 和同样的 instantaneous / prefix code，实现对 XXX 的编码；

换句话说，如果存在一组 uniquely decodeable code 方式，则一定可以找到一组长度 l(x)l(x)l(x) 一一相同的 prefix code 编码方式；此时二者的最短平均编码长度 L1=L2L_1 = L_2L1​=L2​



6.3 最优编码与 Shannon coding


D-adic 的情况：最优编码

可以证明，对于随机变量 XXX ，它对应的最优 prefix codes 满足：

  • ∀x∈X\forall x \in \mathcal{X}∀x∈X，最优编码长度 l∗(x)=log⁡Dp(x)l^*(x) = \log_D p(x)l∗(x)=logD​p(x)​

    • 由于这里的编码长度要是整数，我们将所有满足 ∀x∈X, p(x)=D−n\forall x \in \mathcal{X}, \, p(x) = D^{-n}∀x∈X,p(x)=D−n 的分布称作是 D-adic 的，此时这样的分布能够保证每一个取值 xxx 对应的 l∗(x)l^*(x)l∗(x) 都确实是一个可以取到的整数

    L≥HD(X)L \geq H_D(X) L≥HD​(X)

    当且仅当 p(x)=Dl(x)p(x) = D^{l(x)}p(x)=Dl(x)​ 时 (且分布是 D-adic 时)，等式成立；

香农码与近似最优性质

然而，问题在于不是每一个分布都是 D-adic 的，而 Shannon code 给了一种最粗暴的解决方案，直接向上取整：

**香农码 (Shannon code) **：考虑任意随机变量 XXX ，当进行 DDD 进制编码时 (D-ary alphabet)，可以证明最优编码长度 l∗(x)l^*(x)l∗(x) 和最优平均长度 L∗(x)L^*(x)L∗(x) 满足：

证明：

考虑到香农码计算的时候是直接和 ppp 相关的，也就是说若满足 p1≥p2p_1 \geq p_2p1​≥p2​，则一定有 l1≤l2l_1 \leq l_2l1​≤l2​

换言之如果这里存在一系列编码，则将所有的可能的取值 xxx 按照概率从小到大排序，最小的给最长的编码，依次类推，得到的一定是该编码下最优的香农码；

Ln(X)=1n∑xn∈Xnp(x1,...,xn)l(x1,...,xn)L_n(X) = \frac1n \sum_{x^n \in \mathcal{X}^n} p(x_1, ..., x_n)l(x_1, ..., x_n) Ln​(X)=n1​xn∈Xn∑​p(x1​,...,xn​)l(x1​,...,xn​)
