信息论 (11): 信道容量 channel capacity

发表于2024-04-26|statsinformation theory

笔记基于 Elements of Information Theory (second edition) - Thomas M. Cover, Joy A. Thomas；

其中部分含有个人理解，疏漏还请多指正。


定义一个通信系统；其中信息的传输分成三个步骤：

  • 将原始的信息进行 encoding，得到序列 XXX
  • 进行传输；此时得到的 YYY 是一个和 XXX 相关的随机变量，我们关心的是这个 channel 的概率 p(Y∣X)p(Y|X)p(Y∣X)
  • 对得到的 YYY 进行 decoding，得到最后的信息；

我们重点关心这里的 channel 部分；

本节讨论：

  • channel capacity：信道容量的定义与理解
  • channel coding theorem：定义与证明
  • converse of channel coding theorem

11.1 channel capacity：定义

首先给出定义

换言之，上面定义的 RRR rate 也就是这个 (M,n)(M,n)(M,n) code 的最大可达 rate；

对于一个 memoryless + discrete channel，我们可以定义它的 channel capacity 为：

C=max⁡p(x)I(X;Y)=max⁡p(x)[H(Y)−H(Y∣X)]C = \max_{p(x)} I(X;Y) = \max_{p(x)} [H(Y) - H(Y|X)] C=p(x)max​I(X;Y)=p(x)max​[H(Y)−H(Y∣X)]

11.2 channel coding theorem

考虑 channel capacity 一个最好的性质：当给定的 code rate 满足 R≤CR \leq CR≤C 时，我们可以证明这里的错误率 PeP_ePe​ 是趋向于 0 的；


joint AEP

在这之前我们先定义 joint typicality 和 joint AEP：

回忆前面的 AEP，我们知道长度为 nnn 的序列 xnx^nxn，当 n→∞n \to \inftyn→∞ 时，$ p(x^n) \to 2^{-nH}$，也就是各个序列出现的概率会依概率收敛到常数，此时依概率近似 uniform；

p(xn)→2−nH,−1nlog⁡p(xn)→H(X)p(x^n) \to 2^{-nH}, \quad -\frac1n \log p(x^n) \to H(X) p(xn)→2−nH,−n1​logp(xn)→H(X)

channel coding theorem：证明

channel coding theorem：考虑任意的 discrete + memoryless channel (DMC)，对应的 channel capacity 为 CCC，则任意的 rate R<CR < CR<C 都是可达的 (achievable)：

也就是说能够找到一个 (2nR,n)(2^{nR}, n)(2nR,n) 的 code，对于共 2nR2^{nR}2nR 种可能的原始信息，我们将每一个原始信息编码为长度为 n 的 coding 并通过这个 channel 进行传输 + 解码，可以保证 n→∞n \to \inftyn→∞ 时最大的错误率 λ(n)→0\lambda ^{(n)} \to 0λ(n)→0 ；我们将这样的 code 称作 zero-error code
