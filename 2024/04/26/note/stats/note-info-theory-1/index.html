信息论 (1): 熵，相对熵与互信息

发表于2024-04-26|statsinformation theory

在大 y 的最后一个学期选了 information theory。本是由着给自己找点罪受的原则 (?) 无心之举，但意外这个北方小城的冬春之际迷人得要命，一学期每周二四在 science hill 附近闲逛下来反而有了点醉翁之意不在酒的意思。

言归正传。笔记基于 Elements of Information Theory (second edition) - Thomas M. Cover, Joy A. Thomas；很棒的一本书，算是晦涩里面尽力写得简洁清晰的了。

其中部分含有个人理解，疏漏还请多指正。

先从信息论几个最基本的概念开始：

  • 熵 (entropy)
  • 衍生：联合熵，条件熵
  • 相对熵与互信息

1.1 熵 - Entropy


定义

首先给出熵的定义：

定义 - 熵 (entropy)：考虑一个随机变量 XXX ，取值范围 X\mathcal{X}X，对应分布为 p(x)p(x)p(x) ；则我们定义随机变量 XXX 的熵为：

H(X)=−∑x∈Xp(x)log⁡2p(x)H(X) = -\sum _{x\in \mathcal{X}} p(x) \log_2{p(x)} H(X)=−x∈X∑​p(x)log2​p(x)

  • 熵是随机变量分布 p(x)p(x)p(x) 的函数

也就是说，熵只和分布有关，而和具体的 x 的取值没关系；比如 XXX 以各 1/2 的概率取值为 0 or 1，而 YYY 以各 1/2 的概率取值为 100 or 101，它们的熵的计算都是 1/2+1/2=11/2 + 1/2 = 11/2+1/2=1，是一样的

所以说有时候也可以将随机变量 XXX 的熵写成分布 ppp 的熵 H(p)H(p)H(p)，也就是 H(X)=defH(p)H(X) \xlongequal{def} H(p)H(X)defH(p)​

  • 随机变量 X 的熵 = 随机变量 log⁡1p(X)=−log⁡2p(x)\log {\frac{1}{p(X)}} = -\log_2p(x)logp(X)1​=−log2​p(x) 的期望

同样地，如果把前面的 px 和后面的一串 log 分开

H(X)=∑x∈Xp(x)log⁡21p(x)=Eplog⁡21p(x)\begin{aligned} H(X) &= \sum _{x\in \mathcal{X}} p(x) \log_2 \frac{1}{p(x)} = E_p \log_2 {\frac{1}{p(x)}} \end{aligned} H(X)​=x∈X∑​p(x)log2​p(x)1​=Ep​log2​p(x)1​​

也因为熵是负对数分布的期望，分布是 0-1 之间的，带上负对数以后就是一个正数，所以熵也是非负的，H(X)≥0H(X) \geq 0H(X)≥0

  • 将以 b 为底计算的熵记作 Hb(X)H_b(X)Hb​(X)，满足换底公式：Hb=(log⁡ba)HaH_b = (\log_ba) H_aHb​=(logb​a)Ha​

熵计算中的 log⁡\loglog 的底数并没有严格规定，实际上是可以随意改变的。

这里的对数计算的底和我们需要的进制有关；比如我们知道熵可以理解为 “表示一个随机变量 XXX 的平均最小编码长度"，当我们使用二进制编码的时候（binary），就可以规定以 2 为底 = 用 log⁡2\log_2log2​ 计算，则此时熵计算的结果也就可以理解为：以 bit 作为单位，表示一个随机变量 XXX 平均最小需要几个单位；

当然也可以换其他的底，比如 e 和 10，对于以 b 为底计算的熵，一般记作 Hb(X)H_b(X)Hb​(X)

不同底计算得到的熵可以互相转换；这里用到对数的换底公式：

Hb(X)Ha(X)=Ep(x)(−log⁡bp(x))Ep(x)(−logap(x))=−log⁡bp(x)−logap(x)=log⁡ba\frac{H_b(X)}{H_a(X)} = \frac{E_{p(x)}(-\log_b p(x))}{E _{p(x)}(-log_ap(x))} = \frac{-\log_b p(x)}{-log_ap(x)} = \log_ba Ha​(X)Hb​(X)​=Ep(x)​(−loga​p(x))Ep(x)​(−logb​p(x))​=−loga​p(x)−logb​p(x)​=logb​a

  • 熵的有界性：0≤H(X)≤log⁡∣X∣0 \leq H(X) \leq \log |\mathcal{X}|0≤H(X)≤log∣X∣

首先熵是非负数；H(X)≥0H(X) \geq 0H(X)≥0

因为概率 p(x)∈[0,1]p(x) \in [0, 1]p(x)∈[0,1]，则随机变量 log⁡21p(x)≥0\log _2 \frac{1}{p(x)} \geq 0log2​p(x)1​≥0，它的期望一定也就是 ≥0\geq 0≥0 的

当且仅当 p(x)=1p(x) = 1p(x)=1 时，也就是 XXX 是常数时，对应的熵 = 0

再考虑，随机变量的取值的集合 X\mathcal{X}X，也就是 X∈XX\in \mathcal{X}X∈X，则此时若 XXX 共有 ∣X∣|\mathcal{X}|∣X∣ 种不同的取值，熵满足：

H(X)≤log⁡∣X∣H(X) \leq \log |\mathcal{X}| H(X)≤log∣X∣

当且仅当 XXX 为 X\mathcal{X}X 上的均匀分布时，等式成立；

证明：(这里用到后面的概念相对熵)

考虑一个新的随机变量对应 X\mathcal{X}X 上的均匀分布，记作 u(x)=1/∣X∣u(x) = 1/ |\mathcal{X}|u(x)=1/∣X∣，则此时原分布 p(x)p(x)p(x) 和均匀分布之间的相对熵可以计算为：

D(p∣∣u)=∑p(x)log⁡p(x)u(x)=∑p(x)log⁡p(x)−∑p(x)log⁡∣X∣=H(X)−log⁡∣X∣≥0⇒H(X)≥log⁡∣X∣\begin{aligned} D(p||u) &= \sum p(x) \log \frac{p(x)}{u(x)} \\ &= \sum p(x) \log p(x) - \sum p(x) \log |\mathcal{X}| \\ &= H(X) - \log |\mathcal{X}| \\ &\geq 0 \\ \Rightarrow H(X) &\geq \log |\mathcal{X}| \end{aligned} D(p∣∣u)⇒H(X)​=∑p(x)logu(x)p(x)​=∑p(x)logp(x)−∑p(x)log∣X∣=H(X)−log∣X∣≥0≥log∣X∣​

且当且仅当 p(x)=u(x)⇒uniformp(x) = u(x) \Rightarrow \text{uniform}p(x)=u(x)⇒uniform​ 的时候，等式成立；


理解：熵，不确定性与信息量

可以从两个角度来直观 (但不严谨地) 理解熵：

  • 熵是衡量随机变量分布 不确定性的度量；
  • 熵减等价于信息量；

首先从定义出发；熵本质是分布的函数，是衡量一个随机变量的不确定性的指标；当随机变量的分布更均匀，没有明显的倾向时，我们对随机变量的取值的判断会更不确定，对应的熵会更大；

Tip

example: 均匀分布的熵

考虑一个随机变量的取值有 32 种可能性，每一个都等概率（uniform，也就是任意一个结果的概率都是 1/32），则此时随机变量 XXX 对应的熵为：

H(X)=−32(132log⁡2132)=5H(X) = -32 (\frac{1}{32} \log_2 \frac{1}{32}) = 5 H(X)=−32(321​log2​321​)=5

类似地，考虑这个随机变量的 32 个 outcome 对应的概率变成 (1,0,...,0)(1, 0, ..., 0)(1,0,...,0)，则对应的 XXX 的熵变成：

H(X)=−1log⁡21=0H(X) = -1\log_2 1 = 0 H(X)=−1log2​1=0

对于 uniform 的情况，此时 32 个 outcome 都有可能，且彼此概率差不多，我的不确定性很高 = 熵很大；而对于极端的情况 2，此时可以很笃定 X 的取值一定为 outcome 1，对应的随机性很小，熵更小；


Tip

example: Bernoulli 分布的熵

考虑随机变量 X∼Ber(p)X \sim Ber(p)X∼Ber(p)，也就是以 p 的概率取值为 1，以 1-p 的概率取值为 0，则此时 XXX 的熵为：

H(X)=−plog⁡2p−(1−p)log⁡2(1−p)=defH(p)H(X) = -p\log_2p - (1-p)\log_2 (1-p) \xlongequal{def} H(p) H(X)=−plog2​p−(1−p)log2​(1−p)defH(p)

熵是一个和分布 p(x) 有关的函数，而 Bernoulli 分布被参数化为了一个和 p 有关的函数，则熵也就是一个和 p 有关的函数；当 p = 1/2 的时候熵达到最大（直观理解，也就是 0-1 各自都很有可能 ，我们很难确定到底是哪一个）而 p 接近 1 或接近 0 的时候熵也趋向于 0（直观理解，当 p = 1 的时候我很确定一定 X=1X = 1X=1，此时熵 = 不确定性 = 0；p = 0 的时候同理）

换一个方向，从信息量的角度来说，熵衡量不确定性，而信息带来不确定性的减少，其中造成的熵减也就是信息量的度量；

首先定义信息量：信息量也就是衡量携带信息多少的度量，而信息是用来减少不确定性的。从另一个角度也可以简单理解为，信息量也就是得到信息后能够消除的不确定性的多少，则观测到信息这一行为带来的熵减也就是一种信息量的度量。

类似的，熵和信息量在量上是相等的关系：随机变量 X 的熵度量的是 X 的不确定性，而当我观测到 XXX 随机变量的值时X=xX = xX=x，对于 X 的不确定性也就被消除了，此时观测 X=xX = xX=x​ 这一行为带来了熵减，我得到了等价于 X 的熵的信息量；



1.2 联合熵与条件熵

对于单个随机变量可以定义熵，那现在来考虑一对随机变量 X,YX, YX,Y

联合熵 - joint entropy

考虑存在一对随机变量 X,YX, YX,Y，各自的取值范围为 X∈XX\in \mathcal{X}X∈X， Y∈YY \in \mathcal{Y}Y∈Y，对应联合分布 p(x,y)=P(X=x,Y=y)p(x, y) = P(X=x, Y=y)p(x,y)=P(X=x,Y=y)，则我们可以定义一对随机变量的 联合熵（joint entropy）：

H(X,Y)=−∑X∈X∑Y∈Yp(x,y)log⁡2p(x,y)H(X, Y) = -\sum _{X\in \mathcal{X}} \sum _{Y\in \mathcal{Y}} p(x, y) \log_2 p(x, y) H(X,Y)=−X∈X∑​Y∈Y∑​p(x,y)log2​p(x,y)

  • 联合熵是分布 p(x,y)p(x, y)p(x,y)​​ 的函数

  • 联合熵是随机变量 log⁡21p(x,y)=−log⁡2p(x,y)\log_2 \frac{1}{p(x, y)} = -\log_2 p(x, y)log2​p(x,y)1​=−log2​p(x,y)​ 的期望

类似于对于单个随机变量定义的熵，一样的式子

H(X,Y)=Ep(x,y)log⁡21p(x,y)=−Ep(x,y)log⁡2p(x,y)H(X,Y) = E_{p(x, y)} \log_2 \frac{1}{p(x, y)} = -E _{p(x, y)} \log_2 p(x, y) H(X,Y)=Ep(x,y)​log2​p(x,y)1​=−Ep(x,y)​log2​p(x,y)

  • 类似推广到多元的情况

考虑此时存在 nnn 个随机变量 X1,X2,...,XnX_1, X_2, ..., X_nX1​,X2​,...,Xn​​，则可以定义联合熵

H(X1,...,Xn)=−∑x1,...,xnp(x1,...,xn)log⁡2p(x1,...,xn)H(X_1, ..., X_n) = -\sum _{x_1, ..., x_n} p(x_1, ..., x_n) \log_2 p(x_1, ..., x_n) H(X1​,...,Xn​)=−x1​,...,xn​∑​p(x1​,...,xn​)log2​p(x1​,...,xn​)

类似地，这里的多元的联合熵也就是联合分布 p(X1,...,Xn)p(X_1, ..., X_n)p(X1​,...,Xn​) 的函数；

注意这里的联合熵不大于熵的加和：H(X1,...,Xn)≤∑i=1nH(Xi)H(X_1, ..., X_n) \leq \sum _{i=1}^n H(X_i)H(X1​,...,Xn​)≤∑i=1n​H(Xi​)

证明用到后面会提的熵的链式法则

H(X1,...,Xn)=∑inH(Xi∣Xi−1,...,X1)⇒H(Xi)≥H(Xi∣Xi−1,...,X1)H(X1,...,Xn)≥∑i=1nH(Xi)\begin{aligned} H(X_1, ..., X_n) &= \sum _i ^n H(X_i | X _{i-1} , ..., X_1) \\ \Rightarrow H(X_i) &\geq H(X_i | X _{i-1}, ..., X_1) \\ H(X_1, ..., X_n) &\geq \sum _{i=1}^n H(X_i) \end{aligned} H(X1​,...,Xn​)⇒H(Xi​)H(X1​,...,Xn​)​=i∑n​H(Xi​∣Xi−1​,...,X1​)≥H(Xi​∣Xi−1​,...,X1​)≥i=1∑n​H(Xi​)​

  • 当且仅当 XiX_iXi​ 之间独立的时候等式成立；也就是说对于独立的随机变量，此时联合分布也就是大家的分布的乘积，带上 log 以后变成加和，则大家的联合熵 = 各自的熵的加和；

    H(X1,...,Xn)=∑H(Xi)H(X_1, ..., X_n) = \sum H(X_i) H(X1​,...,Xn​)=∑H(Xi​)

  • 对于二元的情况，联合熵满足：

    H(X,Y)=H(X)+H(Y)−I(X;Y)≤H(X)+H(Y)H(X,Y) = H(X) + H(Y) - I(X;Y) \leq H(X) + H(Y) H(X,Y)=H(X)+H(Y)−I(X;Y)≤H(X)+H(Y)

    当且仅当 X 和 Y 独立的时候等式成立；


条件熵 - conditional entropy

同样地，熵的分布 p(x)p(x)p(x) 可以换成联合分布，那也可以换成条件分布 p(y∣x)p(y|x)p(y∣x)

考察单个随机变量 YYY 的熵，那给定了另一个随机变量 X=xX=xX=x 的时候，Y∣X=xY|X = xY∣X=x 也是一个单个随机变量（它的分布 p(Y∣X=x)p(Y|X = x)p(Y∣X=x) 依赖于 YYY 的分布和常数 xxx ），我们同样可以计算它的熵：

H(Y∣X=x)=p(Y∣X=x)log⁡2p(Y∣X=x)H(Y|X = x) = p(Y|X = x) \log_2 p(Y|X = x) H(Y∣X=x)=p(Y∣X=x)log2​p(Y∣X=x)

但是 XXX 也是随机变量嘛，我将熵 H(Y∣X=x)H(Y|X = x)H(Y∣X=x) 在 XXX 的分布上作期望，得到的也就是 "如果已知了随机变量 XXX 的值，则对应的 Y∣X=xY|X = xY∣X=x 的熵平均下来该是多少"；将它记作 条件熵（conditional entropy）：

H(Y∣X)=∑x∈Xp(x)H(Y∣X=x)H(Y|X) = \sum _{x\in \mathcal{X}} p(x) H(Y|X = x) H(Y∣X)=x∈X∑​p(x)H(Y∣X=x)

一般计算条件熵需要两步：1) 先给定 X=xX = xX=x，计算随机变量 Y∣X=xY | X = xY∣X=x 的熵；2) 对前面计算得到的熵按照 p(x)p(x)p(x) 计算期望

  • 条件熵 H(Y∣X)H(Y|X)H(Y∣X) 可以表示为随机变量 log⁡21p(y∣x)=−log⁡2p(y∣x)\log_2 \frac{1}{p(y|x)} = -\log_2 p(y|x)log2​p(y∣x)1​=−log2​p(y∣x) 在联合分布 p(x,y)p(x, y)p(x,y) 下的期望：

H(Y∣X)=∑x∈Xp(x)H(Y∣X=x)=−∑X∈Xp(x)∑Y∈Yp(y∣x)log⁡2p(y∣x)=−∑x∈X∑Y∈Yp(x,y)log⁡2p(y∣x)=E(−log⁡2p(y∣x))\begin{aligned} H(Y|X) &= \sum _{x\in \mathcal{X}} p(x) H(Y|X = x) = - \sum _{X\in \mathcal{X}} p(x) \sum _ {Y\in \mathcal{Y}}p(y|x) \log_2 p(y|x) \\ &= -\sum _{x\in \mathcal{X}} \sum _{Y \in \mathcal{Y}} p(x, y) \log_2 p(y|x) \\ &= E(-\log_2p(y|x)) \end{aligned} H(Y∣X)​=x∈X∑​p(x)H(Y∣X=x)=−X∈X∑​p(x)Y∈Y∑​p(y∣x)log2​p(y∣x)=−x∈X∑​Y∈Y∑​p(x,y)log2​p(y∣x)=E(−log2​p(y∣x))​

  • **条件熵不是对称的，允许 ∗∗H(Y∣X)≠H(X∣Y)**H(Y|X) \neq H(X|Y)∗∗H(Y∣X)=H(X∣Y)

简单理解，前者代表的是（给定 XXX 的时候，Y∣X=xY|X=xY∣X=x 的平均不确定性），而后者是（给定 YYY 的时候，X∣Y=yX|Y=yX∣Y=y 的平均不确定性）。比如 X 可以是一个很牛逼的变量，它含有了我们判断 Y 所需的所有的信息，此时前者会很小，但是后者偏大

example: 条件熵的不对称

考虑联合分布 p(x,y)p(x, y)p(x,y)

probability X = 1 X = 2 X = 3 X = 4
Y = 1 1/4 1/4 0 0
Y = 2 0 0 1/4 1/4

则此时只要已知了 X 的值，Y 是一定确定的，对应 H(Y|X) = 0，不确定性为 0；而给定了 Y 的时候，X 并不是确定的，H(X∣Y)≠0H(X|Y) \neq 0H(X∣Y)=0

  • 条件一定带来熵的减少：H(X)≥H(X∣Y)≥0H(X) \geq H(X|Y) \geq 0H(X)≥H(X∣Y)≥0

第一个不等式：H(X)−H(X∣Y)=I(X;Y)≥0H(X) - H(X|Y) = I(X;Y) \geq 0H(X)−H(X∣Y)=I(X;Y)≥0

第二个不等式：因为是 −log⁡p(x∣y)-\log p(x|y)−logp(x∣y)​ 的期望，一定是非负的；

理解

  • 当且仅当 XXX 是 YYY 的函数 (也就是 X=f(Y)X = f(Y)X=f(Y)) 的时候等式成立；也就是 Y 一旦确定，按照函数的一一对应则 X=f(Y)X = f(Y)X=f(Y)​ 也就是确定的，此时 Y 可以将 X 的熵直接降低为 0；
  • condition 了之后, 也就是新获得了 Y 的信息后，X 的熵一定是减少 / 不变的；当且仅当 X 和 Y 独立时，等式成立；


1.3 相对熵与互信息

相对熵与互信息可以用来衡量两个分布之间的差异；

相对熵 - relative

考虑存在两个分布 p(x)p(x)p(x) 和 q(x)q(x)q(x)，使用相对熵（relative entropy）/ 也称 KL 散度（Kullback–Leibler distance）来衡量两个分布之间的差距：

D(p∣∣q)=∑x∈Xp(x)log⁡p(x)q(x)\begin{aligned} D(p||q) &= \sum _{x\in\mathcal{X}} p(x) \log \frac{p(x)}{q(x)} \end{aligned} D(p∣∣q)​=x∈X∑​p(x)logq(x)p(x)​​

  • 相对熵 D(p∣∣q)D(p||q)D(p∣∣q) 可以理解成随机变量 log⁡p(x)/q(x)\log {p(x)/q(x)}logp(x)/q(x) 的，在 x∼p(x)x \sim p(x)x∼p(x) 下的期望

D(p∣∣q)=∑x∈Xp(x)log⁡p(x)q(x)=Ep(x)(log⁡p(x)q(x))\begin{aligned} D(p||q) &= \sum _{x\in\mathcal{X}} p(x) \log \frac{p(x)}{q(x)} \\ &= E_{p(x)}(\log \frac{p(x)}{q(x)}) \end{aligned} D(p∣∣q)​=x∈X∑​p(x)logq(x)p(x)​=Ep(x)​(logq(x)p(x)​)​

  • 相对熵的取值是可能发散的

简单的例子，如果存在某一个 x∈Xx\in \mathcal{X}x∈X，使得 p(x)>0p(x) > 0p(x)>0 而 q(x)=0q(x) = 0q(x)=0（简单理解也就是 ppp 分布可以取到这个值，而 qqq 取不到），则此时计算期望的时候会存在一项：p(x)log⁡p(x)0→∞p(x) \log \frac{p(x)}{0} \to \inftyp(x)log0p(x)​→∞，对应的相对熵 / KL 散度也会发散 D(p∣∣q)=∞D(p||q) = \inftyD(p∣∣q)=∞；

  • 相对熵是非负的：D(p∣∣q)≥0D(p||q) \geq 0D(p∣∣q)≥0，当且仅当 p=qp = qp=q 时，等号成立

证明：

直接用到 Jensen 不等式；因为这里的 log⁡\loglog 是凹函数，则满足 E(log⁡X)≤log⁡(EX)E(\log X) \leq \log(EX)E(logX)≤log(EX)

D(p∣∣q)=∑xp(x)log⁡p(x)q(x)=−∑xp(x)log⁡q(x)p(x)=−Ep(x)log⁡q(x)p(x)≥−log⁡Ep(x)q(x)p(x)=−log⁡∑p(x)q(x)p(x)=log⁡1=0\begin{aligned} D(p||q) &= \sum _x p(x) \log \frac{p(x)}{q(x)} \\ &= -\sum _x p(x) \log \frac{q(x)}{p(x)} = -E _{p(x)} \log \frac{q(x)}{p(x)} \\ &\geq -\log E _{p(x)} \frac{q(x)}{p(x)} = -\log \sum p(x) \frac{q(x)}{p(x)} = \log 1 = 0 \end{aligned} D(p∣∣q)​=x∑​p(x)logq(x)p(x)​=−x∑​p(x)logp(x)q(x)​=−Ep(x)​logp(x)q(x)​≥−logEp(x)​p(x)q(x)​=−log∑p(x)p(x)q(x)​=log1=0​

  • 相对熵直观可以理解为两个分布之间的差异程度 / 距离，但并不是严格意义上的一个 metric

    • 它不是对称的：

      D(p∣∣q)≠D(q∣∣p)D(p||q) \neq D(q||p) D(p∣∣q)=D(q∣∣p)

    • 且它不满足三角不等式：

      D(p1∣∣p2)+D(p2∣∣p3) ≱D(p1∣∣p3)D(p_1||p_2) + D(p_2||p_3) \, \ngeq D(p1||p3) D(p1​∣∣p2​)+D(p2​∣∣p3​)≱D(p1∣∣p3)


条件相对熵

类似条件熵，如果我存在两个条件分布 p(y∣x)p(y|x)p(y∣x) 和 q(y∣x)q(y|x)q(y∣x)，则给定 x 的值的时候二者退化为只和 y 有关的分布，可以计算二者的相对熵；然后再基于 x 的值做 "期望"，得到 p(y∣x)p(y|x)p(y∣x) 和 q(y∣x)q(y|x)q(y∣x)​ 两个分布的 KL 散度 / 相对熵

计算还是分成两步：

  • 给定 X=xX = xX=x，得到两个分布 p(Y∣X=x)p(Y|X=x)p(Y∣X=x) 和 q(Y∣X=x)q(Y|X=x)q(Y∣X=x)，计算相对熵；
  • 通过 p(x)p(x)p(x) 对上面计算得到的相对熵做期望；

定义：条件相对熵（conditional relative entropy）：

D(p(y∣x)∣∣q(y∣x))=∑x∈Xp(x)∑y∈Yp(y∣x=x)log⁡p(y∣x=x)q(y∣x=x)=∑x∈X,y∈Yp(x,y)log⁡p(y∣x)q(y∣x)=Ep(x,y)(log⁡p(y∣x)q(y∣x))\begin{aligned} D(p(y|x) || q(y|x)) &= \sum _{x\in\mathcal{X}} p(x) \sum _{y\in \mathcal{Y}}p(y|x=x) \log \frac{p(y|x=x)}{q(y|x=x)} \\ &= \sum _{x\in \mathcal{X}, y\in \mathcal{Y}} p(x, y) \log \frac{p(y|x)}{q(y|x)} \\ &= E_{p(x,y)}(\log \frac{p(y|x)}{q(y|x)}) \end{aligned} D(p(y∣x)∣∣q(y∣x))​=x∈X∑​p(x)y∈Y∑​p(y∣x=x)logq(y∣x=x)p(y∣x=x)​=x∈X,y∈Y∑​p(x,y)logq(y∣x)p(y∣x)​=Ep(x,y)​(logq(y∣x)p(y∣x)​)​

条件相对熵也就是相对熵的一种特殊情况，同样满足

D(p(y∣x)∣∣q(y∣x))≥0D(p(y|x) || q(y|x)) \geq 0 D(p(y∣x)∣∣q(y∣x))≥0

当且仅当 ∀x,y\forall x, y∀x,y，都有 p(y∣x)=q(y∣x)p(y|x) = q(y|x)p(y∣x)=q(y∣x)​ 时，等式成立；


常见分布的相对熵

任意一个分布 ppp 和独立的 X\mathcal{X}X 上的 uniform 的相对熵：

D(p∣∣u)=Ep(x)log⁡p(x)1/∣X∣=Ep(x)log⁡p(x)−Ep(x)log⁡1∣X∣=log⁡∣X∣−H(p)\begin{aligned} D(p||u) &= E _{p(x)} \log \frac{p(x)}{1/|\mathcal{X}|} \\ &= E_{p(x)} \log p(x) - E _{p(x)}\log \frac{1}{|\mathcal{X}|} \\ &= \log|\mathcal{X}| - H(p) \end{aligned} D(p∣∣u)​=Ep(x)​log1/∣X∣p(x)​=Ep(x)​logp(x)−Ep(x)​log∣X∣1​=log∣X∣−H(p)​

正态分布的相对熵：N(μ1,σ2),N(μ2,σ2)N(\mu_1, \sigma^2), N(\mu_2, \sigma^2)N(μ1​,σ2),N(μ2​,σ2)

考虑 f(x)=N(μ1,σ2)f(x) = N(\mu_1, \sigma^2)f(x)=N(μ1​,σ2), g(x)=N(μ2,σ2)g(x) = N(\mu_2, \sigma^2)g(x)=N(μ2​,σ2)，也就只是均值不同的两个正态分布，则此时 f(x) / g(x) 也就是前面的分式抵消，而后面的 exp 上的指数相减；

f(x)g(x)=exp⁡(−((x−μ1)2−(x−μ2)2)/2σ2)log⁡f(x)g(x)=1σ2(μ1−μ2)x−(μ12−μ22)/2σ2D(f∣∣g)=1σ2(μ1−μ2)Ef(x)−(μ12−μ22)/2σ2Ef(x)=μ1⇒D(f∣∣g)=(μ1−μ2)22σ2\frac{f(x)}{g(x)} = \exp(-((x-\mu_1)^2 - (x-\mu_2)^2)/2\sigma^2) \\ \log \frac{f(x)}{g(x)} = \frac{1}{\sigma^2} (\mu_1 - \mu_2)x - (\mu_1^2 - \mu_2^2)/2\sigma^2 \\ D(f||g) = \frac{1}{\sigma^2} (\mu_1 - \mu_2) E_f(x) - (\mu_1^2 - \mu_2^2)/2\sigma^2 \\ E_f(x) = \mu_1 \\ \Rightarrow D(f||g) = \frac{(\mu_1-\mu_2)^2}{2\sigma^2} g(x)f(x)​=exp(−((x−μ1​)2−(x−μ2​)2)/2σ2)logg(x)f(x)​=σ21​(μ1​−μ2​)x−(μ12​−μ22​)/2σ2D(f∣∣g)=σ21​(μ1​−μ2​)Ef​(x)−(μ12​−μ22​)/2σ2Ef​(x)=μ1​⇒D(f∣∣g)=2σ2(μ1​−μ2​)2​

Bernoulli 分布的相对熵：Ber(p),Ber(q)Ber(p), Ber(q)Ber(p),Ber(q)

类似地，考虑 f(x)∼Ber(p)f(x) \sim Ber(p)f(x)∼Ber(p)，g(x)∼Ber(q)g(x) \sim Ber(q)g(x)∼Ber(q)

D(f∣∣g)=plog⁡pq+(1−p)log⁡1−p1−qD(f||g) = p \log \frac{p}{q} + (1-p) \log \frac{1-p}{1-q} D(f∣∣g)=plogqp​+(1−p)log1−q1−p​

D(f∣∣g)=log⁡2−H(X)D(f||g) = \log2 - H(X) D(f∣∣g)=log2−H(X)

注意前面提到过 H(X)H(X)H(X) 的性质，而对于 Bernoulli 分布，也就是在 p = 1/2 的时候会达到最大；此时 H(X) 和 p 的关系应该是类似一个拱门，在 p = 1/2 的时候达到最大；



互信息 - mutual information

类似地，考虑一种特殊情况的相对熵：定义 X,YX, YX,Y 两个随机变量，计算 联合分布p(x,y)p(x, y)p(x,y) 和两个边际分布的乘积 p(x)p(y)p(x)p(y)p(x)p(y) 之间的相对熵，记作 互信息（mutual information）

I(X;Y)=D(p(x,y)∣∣p(x)p(y))=∑X∈X,Y∈Yp(x,y)log⁡p(x,y)p(x)p(y)\begin{aligned} I(X;Y) &= D(p(x, y) || p(x)p(y)) \\ &= \sum _{X\in \mathcal{X}, Y\in \mathcal{Y}} p(x, y) \log \frac{p(x,y)}{p(x)p(y)} \end{aligned} I(X;Y)​=D(p(x,y)∣∣p(x)p(y))=X∈X,Y∈Y∑​p(x,y)logp(x)p(y)p(x,y)​​

类似地，互信息是随机变量 log⁡p(x,y)p(x)q(x)=log⁡p(x∣y)p(x)\log \frac{p(x, y)}{p(x)q(x)} = \log \frac{p(x|y)}{p(x)}logp(x)q(x)p(x,y)​=logp(x)p(x∣y)​ 在联合分布 p(x,y)p(x, y)p(x,y) 下的期望：

I(X;Y)=∑X∈X,Y∈Yp(x,y)log⁡p(x,y)p(x)p(y)=Ep(x,y)log⁡p(x,y)p(x)q(x)=Ep(x,y)log⁡p(x∣y)p(x)\begin{aligned} I(X;Y) &= \sum _{X\in \mathcal{X}, Y\in \mathcal{Y}} p(x, y) \log \frac{p(x,y)}{p(x)p(y)} \\ &= E_{p(x, y)} \log \frac{p(x, y)}{p(x)q(x)} \\ &= E _{p(x, y)} \log \frac{p(x|y)}{p(x)} \end{aligned} I(X;Y)​=X∈X,Y∈Y∑​p(x,y)logp(x)p(y)p(x,y)​=Ep(x,y)​logp(x)q(x)p(x,y)​=Ep(x,y)​logp(x)p(x∣y)​​

互信息非负 I(X;Y)≥0I(X;Y)\geq 0I(X;Y)≥0 ，当且仅当 x 和 y 独立时，互信息为 0

证明：

互信息本质上是相对熵的一种特殊情况，相对熵就是非负的

当且仅当 p(x)=q(x)⇒p(x,y)=p(x)p(y)⇒independentp(x) = q(x) \Rightarrow p(x, y) = p(x)p(y) \Rightarrow \text{independent}p(x)=q(x)⇒p(x,y)=p(x)p(y)⇒independent 的时候等式成立

随机变量 X 自己和自己的互信息 = X 的熵

I(X;X)=Ep(x,x)[log⁡p(x,x)p(x)p(x)]=Ep(x)[log⁡1p(x)]=H(X)\begin{aligned} I(X;X) &= E _{p(x, x)} [\log \frac{p(x, x)}{p(x)p(x)}] \\ &= E _{p(x)}[\log \frac{1}{p(x)}] = H(X) \end{aligned} I(X;X)​=Ep(x,x)​[logp(x)p(x)p(x,x)​]=Ep(x)​[logp(x)1​]=H(X)​

也就是 X 能为 X 提供的信息等价于 X 所有的不确定性，此时互信息 = X 的熵


多元的情况与条件互信息

类似联合熵，我们也可以将互信息的概念推广到多元的情况：

考虑存在三个随机变量 X1,X2,YX_1, X_2, YX1​,X2​,Y，则定义 X1,X2X_1, X_2X1​,X2​ 与 YYY 的互信息为：

I(X1,X2;Y)=H(X1,...,Xn,Y)−H(X1,...,Xn∣Y)I(X_1, X_2 ;Y) = H(X_1, ..., X_n, Y) - H(X_1, ..., X_n|Y) I(X1​,X2​;Y)=H(X1​,...,Xn​,Y)−H(X1​,...,Xn​∣Y)

从图上来看，I(X1,X2;Y)I(X_1, X_2 ; Y)I(X1​,X2​;Y)对应的也就是绿色部分，表示 X1 和 X2 合在一起时能够提供的有关于 YYY 的信息；

$$ I(X; Y, Z) = D(P _{x, y, z} || P_{x}P_{Y, Z}) \\ $$

类似地，也可以在给定了另一个随机变量 ZZZ 的情况下，再讨论两个随机变量 X,YX, YX,Y 之间的互信息。定义 Z 已知时，随机变量 X 和 Y 的互信息为 条件互信息（conditional mutual information） 为条件联合概率 p(x,y∣z)p(x, y|z)p(x,y∣z) 和 p(x∣z)p(y∣z)p(x|z)p(y|z)p(x∣z)p(y∣z) 之间的相对熵：

I(X;Y∣Z)=∑zp(z)∑p(x,y∣z)log⁡p(x,y∣z)p(x∣z)p(y∣z)=Ep(x,y∣z)log⁡p(x,y,z)p(x∣z)p(y∣z)\begin{aligned} I(X;Y | Z) &= \sum_z p(z)\sum p(x, y|z) \log \frac{p(x, y|z)}{p(x|z)p(y|z)} \\ &= E_{p(x, y | z)} \log \frac{p(x, y,z)}{p(x|z) p(y|z)} \end{aligned} I(X;Y∣Z)​=z∑​p(z)∑p(x,y∣z)logp(x∣z)p(y∣z)p(x,y∣z)​=Ep(x,y∣z)​logp(x∣z)p(y∣z)p(x,y,z)​​

  • 类似地，条件互信息 I(X;Y∣Z)I(X;Y|Z)I(X;Y∣Z)也就是随机变量 log⁡p(x,y∣z)p(x∣z)p(y∣z)\log \frac{p(x, y|z)}{p(x|z)p(y|z)}logp(x∣z)p(y∣z)p(x,y∣z)​ 的期望

  • 类似条件相对熵，条件互信息也是非负的；当且仅当 p(x,y∣z)=p(x∣z)p(y∣z)p(x, y|z) = p(x|z)p(y|z)p(x,y∣z)=p(x∣z)p(y∣z)，也就是 x 和 y 对 z 条件独立时，等式成立

I(X;Y∣Z)≥0I(X;Y|Z) \geq 0 I(X;Y∣Z)≥0

满足马尔可夫性时, "条件" 一定会带来互信息的减少

假设 X→Y→ZX \to Y \to ZX→Y→Z , 则满足

I(X;Y∣Z)≤I(X;Y)I(X;Y|Z) \leq I(X;Y) I(X;Y∣Z)≤I(X;Y)

证明 refer to data-processing inequity

Warning

不满足 Markov property 的时候，对互信息做 condition 后不一定会出现下降

举个例子, 此时 X,YX, YX,Y 都是概率为 1/2 的 0-1 变量, 且 XY 之间独立, 则 I(X;Y)=0I(X;Y) = 0I(X;Y)=0, 但是当 Z=X+YZ = X + YZ=X+Y​ 已知时, 比如我已知 Z = 1, 则 如果 X = 0, Y 一定就是 1; 而如果 X = 1, Y 一定就是 0; 也就是说 Z 已知时, X 的信息反而能够直接帮助我们确定 Y 的取值, 此时二者是强相关的

I(X;Y)=0I(X;Y∣Z)=H(X∣Z)−H(X∣Y,Z)=H(X∣Z)=p(Z=0)H(X∣Z=0)+p(Z=1)H(X∣Z=1)+p(Z=2)H(X∣Z=2)=p(Z=1)H(X∣Z=1)when Z = 0 or 2, X = 0 or 1 with conditional entropy = 0=1/2(−1/2log⁡21/2−1/2log⁡21/2)=1/2≥0\begin{aligned} I(X;Y) &= 0 \\ I(X;Y|Z) &= H(X|Z) - H(X|Y, Z) \\ &= H(X|Z) \\ &= p(Z=0)H(X|Z=0) + p(Z=1)H(X|Z=1) + p(Z = 2)H(X|Z=2) \\ &= p(Z=1)H(X|Z=1) \quad \text{when Z = 0 or 2, X = 0 or 1 with conditional entropy = 0} \\ &= 1/2 (-1/2\log_21/2 - 1/2\log_21/2) \\ &= 1/2 \geq 0 \end{aligned} I(X;Y)I(X;Y∣Z)​=0=H(X∣Z)−H(X∣Y,Z)=H(X∣Z)=p(Z=0)H(X∣Z=0)+p(Z=1)H(X∣Z=1)+p(Z=2)H(X∣Z=2)=p(Z=1)H(X∣Z=1)when Z = 0 or 2, X = 0 or 1 with conditional entropy = 0=1/2(−1/2log2​1/2−1/2log2​1/2)=1/2≥0​


互信息与熵的关系

互信息实际上衡量的是边际熵和条件熵之间的差值：

I(X;Y)=H(X)−H(X∣Y)=H(Y)−H(Y∣X)\begin{aligned} I(X;Y) &= H(X)-H(X|Y) \\ &= H(Y) - H(Y|X) \end{aligned} I(X;Y)​=H(X)−H(X∣Y)=H(Y)−H(Y∣X)​

也就是说，I(X;Y)I(X;Y)I(X;Y) 也可以理解成：当给定了随机变量 YYY 的信息后，我对于随机变量 XXX 的不确定性可以下降多少；直观来说也就是衡量 YYY 可以提供多少有关 XXX 的信息；

证明：直接从随机变量的期望的角度来证：

I(X;Y)=Ep(x,y)−log⁡p(x,y)p(x)p(y)=−Ep(x,y)(log⁡p(x∣y)−log⁡p(x))=Ep(x,y)log⁡p(x)−Ep(x,y)log⁡p(x∣y)=H(X)−H(X∣Y)\begin{aligned} I(X;Y) &= E_{p(x, y)}-\log \frac{p(x, y)}{p(x)p(y)} \\ &= -E_{p(x, y)} (\log p(x|y) - \log p(x)) \\ &= E_{p(x, y)}\log p(x) - E_{p(x, y)} \log p(x|y) \\ &= H(X) - H(X|Y) \end{aligned} I(X;Y)​=Ep(x,y)​−logp(x)p(y)p(x,y)​=−Ep(x,y)​(logp(x∣y)−logp(x))=Ep(x,y)​logp(x)−Ep(x,y)​logp(x∣y)=H(X)−H(X∣Y)​

由于互信息是对称的，同理可以得到 I(X;Y)=H(Y)−H(Y∣X)I(X;Y) = H(Y) - H(Y|X)I(X;Y)=H(Y)−H(Y∣X)

类比，条件互信息其实也就是在衡量两个条件熵之间的差值

I(X;Y∣Z)=H(X∣Z)−H(X∣Y,Z)I(X;Y|Z) = H(X|Z) - H(X|Y,Z) I(X;Y∣Z)=H(X∣Z)−H(X∣Y,Z)

可以理解为：

  • 首先从 XXX 中挖掉 Z 能够提供的信息，考察已知 Z 后 X 还剩下的不确定性 = H(X∣Z)H(X|Z)H(X∣Z)
  • 再考虑，同时已知 Y 和 Z 的时候 X 还剩下的不确定性 = H(X∣Y,Z)H(X|Y, Z)H(X∣Y,Z)
  • 则条件互信息 I(X;Y∣Z)I(X;Y|Z)I(X;Y∣Z) 也就是，在已知了 Z 的情况下，Y 还能够让 X 的不确定性减少多少，还能提供多少关于 X 的新的信息
