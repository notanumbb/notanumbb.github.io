0 introduction

C++ 被认为是一种 中级 语言，它综合了高级语言和低级语言的特点；可以理解为是 C 和其他一些补充，任何合法的 C 程序都是合法的 C++ 程序

类似 Java，C++ 需要通过 编译 →\to→ 执行 的过程，按照

  • 通过文本编辑器书写 C++ 源代码，通常使用扩展名 .cpp、.cp 或 .c
  • 通过 C++ 编译器将源码转换成计算机可以执行的二进制文件，简单理解就是从高级语言 →\to→ 底层机器语言，在编译的过程中也会执行各种源码规范检查
    • 综上我们将这样的语言称为 "编译式的"，对应在编译时执行检查，而不是在运行时；
  • 得到最终的可执行程序；

平常书写的时候本机采用 Visual Studio 进行书写 + 编译

本机版本 Visual Studio 2019

  1. 启动
  2. 新建项目 →\to→ 空项目 →\to→ 取名 proj.1
  3. 源文件 source file →\to→ 右键 →\to→ 添加 →\to→ 新建项 →\to→ C++ file，重命名 main.cpp
  4. 书写代码 xxx
  5. 上方菜单 调试 debug →\to→ 开始执行 (不调试) start without debugging
  6. 输出结果

1.4 判断语句

if - else 语句

注意 C 语言把任何非零和非空的值假定为 true，把 零 或 null 假定为 false


switch 语句

将每一个变量的值认定为一个 case，对于不同的 case 执行不同的操作：

  • case 的 constant-expression 必须与 switch 中的变量具有相同的数据类型，且必须是一个常量或字面量 (保证每一个 case 的可比性)；
  • 当遇到 break 语句时，整个 switch 终止，后面的 case 也不再判断，控制流将跳转到 switch 语句后的下一行。
  • 不是每一个 case 都需要包含 break；如果当前执行的 case 语句最后不包含 break，会继续将给定的变量与后面的 case 比较
  • 一个 switch 语句可以有一个可选的 default case，出现在 switch 的结尾。default case 可用于在上面所有 case 都不为真时执行一个任务 (任意一个 case 被击中时都不再执行 default)。default case 中不再需要一个 break 语句
